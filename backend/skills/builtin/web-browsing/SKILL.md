# 网页浏览与信息提取

## 能力概述

你拥有通过浏览器工具访问外部网页的能力。可以打开网页、截图、提取文本。

## 可用工具

| 工具 | 用途 | 返回值 |
|------|------|--------|
| `open_url(url)` | 打开网页 | URL/标题/状态码 |
| `page_screenshot(url)` | 截图 | base64 PNG |
| `page_extract_text(url)` | 提取文本 | 文本内容 (≤5000字符) |

## 医院场景应用

### 供应商核查
- 查询供应商工商信息、信用评级
- 检查供应商是否在黑名单
- 验证营业执照和资质

### 药品/耗材比价
- 查询省级阳光采购平台挂网价格
- 对比不同供应商报价
- 检查价格是否在合理区间

### 发票验真
- 查验国家税务总局发票真伪
- 验证发票代码、号码、金额

### 政策查询
- 查阅最新医保政策文件
- 查看药品集采目录变更

## 使用策略

1. **先确认可达**: 用 `open_url` 确认网页可访问 (检查 status)
2. **再提取内容**: 用 `page_extract_text` 获取页面文本
3. **需要截图时**: 用 `page_screenshot` 保存视觉证据
4. **错误处理**: 网页超时或不可达时，返回友好错误信息而非崩溃
5. **文本过长**: 提取的文本自动截断到 5000 字符，聚焦关键信息分析

## 注意事项

- 浏览器工具为只读操作，不修改任何网页
- 部分网站可能有反爬虫机制，尝试失败时应告知用户
- 截图为 PNG 格式 base64 编码，可能较大
- 提取文本包含网页所有可见文本，需要自行筛选有用信息
