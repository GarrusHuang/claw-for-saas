# 医院财务体系基础

## 组织层级与审批权限
- 医院财务管理分为：院级、科室级、个人级
- 审批权限按职级划分：科主任 → 分管院长 → 院长 → 财务委员会
- 金额越大，审批层级越高

## 预算管理
- 科室年度预算由财务处统一分配
- 差旅费、办公费、科研费等分项管理
- 预算执行率是科室年度考核指标之一

## 费用报销基本原则
- 实报实销，不得超标
- 必须有合规票据（发票、行程单等）
- 公务支出须事前审批
- 费用标准按职级和城市等级区分

## 合同管理基本原则
- 合同金额与审批权限挂钩
- 供应商须通过资质审核
- 重大合同需法务审核
- 合同执行进度须定期跟踪
