# PDF 文档分析能力

## 概述

Agent 接收到的材料可能包含 PDF 文档的解析结果（文本提取、表格数据、页面结构等）。
本 Skill 提供 PDF 结构化分析策略，帮助 Agent 准确理解文档内容并提取关键信息。

## 多页文档处理

### 页面定位策略
- 关键信息通常集中在首页和末页
- 首页：标题、编号、日期、甲乙方信息
- 末页：签章、总金额、生效日期
- 中间页：正文条款、明细表格

### 跨页内容识别
- 表格跨页：检查连续页面是否有相同列结构的表格
- 段落跨页：上页末尾无句号/分号时，与下页首段拼接
- 条款编号连续性：如第 3 页末尾是"第五条"，第 4 页开头应为"第六条"

### 页面类型判断

| 页面特征 | 类型 | 处理方式 |
|---------|------|---------|
| 包含"合同"/"协议"标题 | 封面页 | 提取标题、编号、日期 |
| 大量"第X条"格式 | 条款页 | 逐条提取 |
| 行列对齐的数据 | 表格页 | 表格结构化提取 |
| 包含签名/印章区域 | 签章页 | 提取签署方和日期 |
| 包含附件/附录标注 | 附件页 | 关联到主文档 |

## 表格提取策略

### 表格识别特征
- 具有规律性的行列对齐
- 存在表头行（通常为首行加粗或有边框）
- 数值列通常右对齐
- 合计行通常在表格末尾

### 表格数据结构化
```
输入（OCR 文本）:
  品名    规格    数量    单价      金额
  手套    M号     100     2.50    250.00
  口罩    N95      50    15.00    750.00
  合计                           1000.00

输出（结构化）:
  headers: [品名, 规格, 数量, 单价, 金额]
  rows: [
    [手套, M号, 100, 2.50, 250.00],
    [口罩, N95, 50, 15.00, 750.00]
  ]
  summary: {合计: 1000.00}
```

### 表格校验
- 数量 x 单价 = 金额（逐行验证，使用 calculator 工具）
- 各行金额之和 = 合计金额（使用 calculate_sum 工具）
- 表头列数与数据行列数一致

## 页眉页脚处理

### 识别策略
- 页眉：每页顶部重复出现的文本（如文档标题、机构名称）
- 页脚：每页底部重复出现的文本（如页码、日期、保密声明）
- 判定标准：连续 3 页以上出现相同或高度相似的文本

### 处理方式
- 页码提取：用于验证文档完整性（是否缺页）
- 重复内容去除：分析正文时忽略页眉页脚
- 保密标注提取：如有"内部"/"机密"标注，记录到文档元数据

## 合同条款识别

### 条款结构模式
```
一级条款：第一条、第二条... 或 一、二、三...
二级条款：1.1、1.2... 或 （一）（二）...
三级条款：(1)、(2)... 或 a)、b)...
```

### 关键条款定位

| 条款类型 | 识别关键词 | 审计重点 |
|---------|-----------|---------|
| 合同标的 | 标的物、服务内容、工作范围 | 明确性、完整性 |
| 金额条款 | 合同金额、总价、单价 | 大小写一致性 |
| 付款条款 | 付款方式、付款期限、预付 | 预付比例是否合规 |
| 违约条款 | 违约责任、违约金、赔偿 | 对等性 |
| 保密条款 | 保密义务、保密期限 | 是否有保密要求 |
| 争议解决 | 仲裁、诉讼、管辖 | 管辖地是否有利 |
| 有效期 | 生效日期、终止日期、有效期 | 期限合理性 |

### 条款完整性检查
- 必备条款清单逐项比对
- 缺失条款标记为审计 warning
- 条款间逻辑一致性检查（如付款总额 = 合同金额）

## 文档分节边界检测

### 分节标志
- 明确的标题行（字号较大、加粗、居中）
- 编号重新起始（从"一"或"1"重新开始）
- 分页符（新章节通常从新页开始）
- "附件"/"附录"/"补充协议"等关键词

### 分节处理策略
1. 识别所有分节标志
2. 为每个节建立索引（节标题 + 起始页/行）
3. 分析时按节处理，避免跨节误读
4. 附件与正文分开处理，附件内容辅助正文理解

## 文档类型快速判断

根据首页特征快速判断文档类型：

| 首页特征 | 文档类型 |
|---------|---------|
| "合同"/"协议"标题 + 甲乙方 | 合同文档 |
| "报销单"/"费用报销" | 报销申请 |
| "发票"/"Invoice" | 发票扫描件 |
| "审批表"/"审批单" | 审批文件 |
| "会议纪要"/"会议记录" | 会议文档 |
| "招标文件"/"投标书" | 招投标文件 |
| "预算"/"经费申请" | 预算文件 |

## 注意事项

- PDF 解析结果可能因扫描质量不同而差异巨大
- 扫描件（图片型 PDF）的文本提取依赖 OCR，准确率较低
- 原生 PDF（文字型）的提取准确率较高
- 表格中的数值计算必须使用 calculator 工具验证
- 关键信息建议多源交叉验证（如合同金额出现在多个位置时比对一致性）
