# 报销业务知识

## 差旅报销核心规则

### 住宿费
- 按出差人员职级和出差城市确定标准
- 需提供住宿发票原件
- 无住宿发票不予报销住宿费
- 多人同住可合并开票，按人均不超标

### 交通费
- 飞机：副高级及以上可乘坐经济舱，正高级可乘商务舱
- 高铁/动车：副高级及以上可乘坐一等座，其他二等座
- 出租车：原则上仅限机场/车站接送，需注明事由

### 餐费补贴
- 按出差城市发放固定补贴，不需要票据
- 接待用餐已报销的不再发放餐补

### 其他费用
- 会议注册费：凭会议通知和发票报销
- 行李托运：经济舱标准内可报销

## 常见场景处理策略

### 场景1：标准差旅
- 单人出差，住宿+交通+餐补
- 先查用户职级 → 查对应城市标准 → 逐项比对

### 场景2：超标报销
- 实际费用超出标准
- 需要标注超标原因
- 审计标记为 warning 而非直接 error（如有合理说明）

### 场景3：多人出差
- 同一行程多人
- 住宿可能共用一间
- 交通分别计算

## 关键注意事项
- 所有数值比较必须使用 calculator 工具，严禁心算
- known_values（如报销人姓名、科室）不可修改
- 日期范围决定天数，影响住宿费和餐补总额
- 先获取用户信息，再查费用标准，最后做数值比较
