# 合同业务知识

## 合同分类
- 采购合同：设备、耗材、药品采购
- 服务合同：维修维保、IT 服务、咨询服务
- 工程合同：基建、装修、改造
- 科研合同：合作研究、技术转让

## 合同起草核心要素
- 合同名称：明确合同类型和标的
- 甲乙双方：全称 + 统一社会信用代码
- 合同标的：品名、规格、数量、单价
- 合同金额：大小写一致
- 付款方式：预付款比例不超过30%
- 履约期限：明确起止日期
- 违约条款：违约金比例通常5-20%
- 争议解决：仲裁或诉讼

## 合同审核要点
- 金额合理性：与市场价格比较
- 付款条件：预付比例、付款节点
- 违约条款：对等性审查
- 知识产权：归属约定
- 保密条款：必要性审查
- 争议管辖：有利于己方

## 审批流程
- 10万以下：科主任审批
- 10-50万：分管院长审批
- 50-100万：院长审批
- 100万以上：院务会审议

## 关键注意事项
- 合同金额必须使用 calculator 工具验证
- 付款比例计算必须使用 calculate_ratio 工具
- known_values 中的合同模板不可修改核心条款
- 供应商信息须通过 MCP 工具查询验证
