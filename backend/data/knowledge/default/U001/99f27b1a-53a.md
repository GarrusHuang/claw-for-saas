# 第三十一章：命令是撤

频道安静了五秒。

五秒很长。异潮打了一整夜，调度频道从来没安静过这么久。哪怕最密集的时候也有赵姐的嘴在转，有哪个组在喊。

五秒里没人说话。

二组组长从路灯杆那边站起来了。脸还是白的，但跟刚才不一样——刚才是“做了决定又撤了”那种白，现在是松了。省局命令放弃，他退的那个决定变成了对的。

“那我们——”他开口。

姜北没看他。

姜北站在巷口，两只脚踩在原来的位置，没动过。他听到命令的时候正要说“消防通道进”，嘴合上了，但身体的朝向没变。还是面对着巷子里面。面对着那栋楼。

沈擎靠在墙上看那栋楼。

六层。灰扑扑的。路灯残余的光只照到三楼以下，四楼往上是暗的。但五楼有一个窗户亮着——不是灯。是手机屏幕。有人在五楼举着手机。

光很小。隔着整条巷子和楼前空地看过去只是一个微弱的白点，一闪一闪的，在晃。

拿手机的人手在抖。

马大壮把嘴里剩的半口压缩饼干咽了。脖子上的筋鼓了一下。

“操。”

一个字。说完把铁管从肩上放下来，攥在手里。不是准备打的姿势。是攥着。

调度频道里钱旺的声音又出来了。不是对三组说的。他在部署全局收缩——“一组维持北线不变。二组撤回清河路设二道防线。城南正面我在。”

没提三组。

钱旺刚才已经说了“东线放弃固守”。三组在东线。命令已经覆盖了。

巷子里的嘶吼声还在。隔着楼板传出来，压着的，分不清几楼。楼前空地上那几只二类在游荡，灰褐色的影子在碎玻璃和灰色液体之间来回踱。

“唐遇。”

姜北开口了。声音不大。

唐遇站在巷口另一侧。终端抱在胸口，眼睛闭着，鼻血干了一层又渗了一层。他听到姜北叫他，没睁眼。

“再扫一遍。”

手指微张。掌心朝着那栋楼的方向。

三秒。

“楼内异种没有变化。二类三只，三到四楼。三类一只，四楼。”

跟刚才一模一样的数据。语速回到了正常——快、平、一字不含糊。

“居民。”

“十七个。五楼。”

这次没有重复。该停的地方他停了半秒，但没重复。

然后他说了一句刚才没说过的话。

“有小孩。”

语速还是平的。报数据的语速。但这三个字不是数据。唐遇报数从来只报数字——几只、几个、什么方位、什么距离。他不报“有小孩”。小孩是人，不是类别，声呐分不出年龄，只分得出信号大小和频率。

他从信号里读出了什么别的东西。

沈擎看着他。

唐遇的手指还张着。掌心朝前。鼻血从左鼻孔挂下来一条细线，到了上唇被干血糊住了，没往下淌。

“最小的那个——”

他停了一下。

不是刚才报居民数时那种停。那种停是数据和情绪之间的缝隙。这一次是他在听。声呐还开着，他在感知那个信号。

“心跳特别快。”

四个字出来之后唐遇自己顿了一下。

不长。比一秒短。但他的手指收了一点，又张开了。像是刚才那四个字从嘴里出来的时候他没准备，嘴比脑子快了。

声呐感知的不是声音。不是心跳的咚咚声。是那个小信号源的生物频率波动——频率比其他十六个高。高出来的那一截，翻译成人话就是心跳特别快。

害怕的时候心跳会快。

小孩的心跳本来就比成年人快。害怕的小孩会更快。

唐遇报了一夜的数据。几百只异种，十几波次，方向、距离、速度、数量。那些数字从他嘴里出来的时候跟读表没区别。他的声呐把一切变成信号，信号没有温度。

但这一次他说了“心跳特别快”。

他不需要说这个。报“十七个，五楼，有小孩”就够了。最小的那个心跳快不快，对战术部署没有任何影响。进楼路线不会因为一个小孩的心跳而改变，分工不会，优先级不会。

他说了。

马大壮攥铁管的手紧了一下。他看了唐遇一眼，目光又转向姜北。

沈擎没看任何人。他在看五楼那个手机屏幕的光。还在亮着。还在晃。

巷子里又传来一声嘶吼。三类的。从楼里传出来，低频闷振。楼前空地的二类听到了，有两只朝单元门方向挪了几步。

频道里赵姐的声音传过来。调度台常规确认。

“三组，收到省局指示了吗？确认一下。”

没人按频道。

两秒。

赵姐又叫了一声：“三组？”

姜北看了沈擎一眼。

沈擎靠在墙上。左肩撑着，省力。短刀插在腰后，虎口破的那只手垂着，没有回响。膝盖磕的，肩膀伤的，补剂剩九支。从城南打到东线，该用的都用了，手里什么都没有。

四只异种。十七个居民。四个带伤的人。消防通道能进，一到三楼窗户完好说明下面干净，真正要打的是四楼往上。三类在四楼。居民在五楼。中间只隔一层。

算不出来赚不赚。但算得出来进去之后怎么走。

他回看姜北。

没点头也没摇头。就是看着。

姜北又看了马大壮。

马大壮攥着铁管，嘴闭着。他不说话的时候下颌骨线条很硬，腮帮子的肌肉绷着。他也看着姜北。

姜北看唐遇。

唐遇睁眼了。手从终端上放下来。鼻血、干血、裂口。他什么都没说。

四个人在巷口站着。二组的人在后面，组长不知道什么时候又蹲下去了。巷子里嘶吼声不断。五楼那个手机屏幕的光还在。

没人说撤。

姜北按了专频。

“三组留下。执行就近救援。”

频道里安静了一秒。

赵姐的声音冒出来。她张了下嘴——沈擎从气流声听出来的，她吸了口气准备说什么。

没说。

两秒后她的声音出来了，跟平时不一样，很轻。

“……收到。”

松开专频。扭头看了一眼巷子。

“消防通道。从背面进。”

他迈步了。两条腿还在抖，步幅比正常短一截，但方向很确定。往巷口右侧那条窄路拐过去——沈擎刚才侦察走的那条。

沈擎从墙上撑起来。左肩动了一下，里面有东西错着磨了一圈，酸胀的，往骨头深处钻。

跟上了。

马大壮跟在后面。铁管换了个握法，右手在前左手在后。

唐遇最后一个。终端抱在胸口，闭上眼，声呐重新铺开。

四个人拐进窄路。两侧墙上小广告翘着角，脚底碾到碎砖。风从楼那边灌过来，带着异种体液的酸腥味，比巷口浓了一层。

前面是楼。

【第三十一章回顾】
梗概：省局命令放弃东线后，频道安静五秒。二组组长松了口气，姜北一动没动。沈擎看到五楼有人用手机照亮——手在抖。马大壮一个字“操”。姜北让唐遇再扫一遍。唐遇报完常规数据后加了“有小孩”——声呐不分年龄，他从信号频率里读出来的。然后说了“最小的那个……心跳特别快”——唐遇从来不报这种信息。四步情感递进完成（第25章“都往这边来了”→第29章握终端→第30章重复“十七个”→本章“心跳特别快”）。赵姐频道确认三组是否收到命令，姜北没回应。四个人对视，没人说撤。姜北按专频：“三组留下。执行就近救援。”赵姐沉默两秒后只说了“收到”。四人从消防通道方向进楼。
伏笔：①唐遇预震④完成——“心跳特别快”。从数据机器到感知数字背后的人，四步递进收束。他自己顿了一下——嘴比脑子快了。②姜北抗命——“姜北选人不选规定”伏笔正式回收。从第17章为老太太停车三分钟→第21章防护服划掉姓名→本章抗命救人，一条完整的性格线。③赵姐反应——“收到”说得很轻，跟平时不一样。她选择了配合而不是阻拦。④五楼手机光——有人在用手机照亮。手在抖。具体的视觉细节让“十七个人”从数字变成画面。⑤二组组长对比——命令下来后他松了口气。与三组的选择形成对照。
状态变化：三组决定违抗命令留下救人。开始从消防通道进楼。回响仍空仓，补剂仍九支。四人全员带伤/透支。赵姐已知三组抗命（“收到”=默认配合）。
下章衔接：第32章“别开枪，我没编制”——进楼。钱旺给二十分钟。消防通道进，楼道漆黑。逐层清理，踹墙绕后。
