# 第三章：这东西会躲

车是一组的。

三组没车。准确说三组有车，一辆面包车，上个月左后轮轴承坏了，报修单走到装备科，至今没批下来。平时出任务要么蹭一组的车，要么蹭二组的车，要么坐公交——真坐过，二类异种在城东菜市场出没那次，沈擎刷的交通卡。

今天不用蹭。一组出动，三组配合，两辆车一起走。孙毅的人开前面那辆，姜北坐后面这辆副驾。沈擎、马大壮、唐遇挤后排。

马大壮占了一半的座。唐遇被挤在车门边上，脸贴着玻璃，面无表情。

“让让。”沈擎拿胳膊肘怼马大壮。

“没地方了！”

“你往那边坐坐怎么了？”

“那边是唐遇！你挤他试试？”

唐遇没回头，隔着玻璃说了句什么。车窗关着，没听清。

姜北回头看了一眼。不是看他们挤——他在数人头。三组四个全到了。回头拿起对讲机。

“三组就位。”

对讲机滋啦响了一下。孙毅的声音：“一组就位。城西工业园东门进。”

后面还跟了一辆黑别克。周鹤的车，不远不近吊着，像来观光的。

城西工业园离分局十二公里，路上车不算多，开了大概二十分钟。沈擎一路没说话，闭着眼睛靠座椅。没睡着。上午搬了十二趟桌子，胳膊酸，但脑子清醒得很。

手里空的。

他下意识攥了攥拳。不是紧张。是那种木匠出门忘带工具箱的感觉。

到了。

城西工业园是九十年代建的，大片厂房，一半停工了，剩下的勉强运转。围墙缺了好几段，杂草从裂缝里长出来半人高。监测组的人已经先到了，三个人蹲在东门口抽烟，看见车来了才站起来。

“信号在3号厂房片区，”监测组组长指了个方向，夹烟的手有点抖，“最后一次脉冲是六分钟前。强度……不太像二类。”

姜北：“确认了吗？”

“设备显示三类置信度百分之七十二。但波形有点怪，不是常见的三类特征。”

孙毅已经在前面布置人了。一组六个人散开，两人一小组，从东门朝3号厂房方向推进。动作规范，科班味儿足——间距、手势、推进速度，像教科书。

姜北没急着动。站在东门口，看了看厂区地形。

3号厂房在园区中段。过去要穿一段空地，两边是废弃的仓库。路面碎石多，草丛齐腰。厂房本身是钢结构顶棚，三面墙，西面敞口。

“唐遇。”

唐遇闭上了眼。大概五六秒，睁开。

“3号厂房里面。一个。不动。”

“大小？”

“……大。”唐遇顿了一下，“比正常三类的信号要宽。”

姜北看了沈擎一眼。沈擎读懂了——不是“你有什么想法”，是“你知道情况了”。

“三组跟在一组后面，保持五十米。沈擎右翼外围，马大壮左翼，唐遇跟我。”

沈擎：“我外围？”

“你手里没东西。”

这话说得直白。回响空仓的意思，姜北没给他面子也没拐弯。

沈擎嘴角撇了一下：“行，站外围，熟练工。”

姜北没接。

马大壮小声嘟囔了一句“给他把刀啊”，从腰后抽出备用的制式短刀，刀鞘都磨毛了，往沈擎手里一塞。

“借你的。”

“回头还我。磨刀石我刚买的，五块钱。”

推进。

空地上碎石踩着咯吱响。太阳还没爬到正头顶，厂房影子拖得老长。风从敞口那面灌进来，带着铁锈味和一股说不上来的腥——淡，但沈擎闻出来了。常年跟异种打交道，鼻子都灵，这股腥味是三类体液挥发。浓度不高，说明它没受伤，或者待的时间不长。

一组先到了厂房外围。孙毅的人在西面敞口两侧蹲下，手势比划了一圈。

里面。

沈擎在右翼位置，隔了三四十米，视线被一堵半塌的矮墙挡了一半。他能看到厂房的钢架顶棚和里面堆着的设备轮廓，看不清深处。

唐遇的声音从对讲机里传出来，压得很低：“还在原位。不动。”

孙毅带人从西面敞口进去了。

标准的三面包抄——两组从两侧推进，一组正面压。手电光扫进去，在钢架之间切来切去。

忽然前面一组的人顿住了。手电光定在一个位置，不动了。

对讲机里有人吸了口气。

然后是孙毅的声音，压着但能听出变调：“……它不在那儿。”

“什么？”姜北。

“有体液痕迹。大滩。但东西不在。”

唐遇猛地睁开眼：“它动了——往北。速度很快。”

北面是两排废弃仓库之间的夹道。一组有两个人刚经过那里。

“一组北面小心——”

对讲机里传出金属被撞变形的闷响。然后是人喊。

沈擎站在矮墙后面，听见了。不是人被打的声音——是仓库铁皮墙被什么东西从里面撞凸出来的声音。它在夹道里等着。一组的人经过夹道口，它没冲出来。等人走过去了，从侧面撞墙。

铁皮墙变形的位置离那两个人不到三米。吓唬的。它没真打，就是砸了一下墙，趁人慌的那半秒，从夹道另一头跑了。

“它在试我们。”唐遇的声音。

姜北已经在跑了。“一组两人无伤。它往东去了。”

三类异种。但这个行为模式不对。

沈擎跟过低等级异种，也跟过正常三类。二类是纯粹的攻击本能，看见人就冲。正常三类有基本的趋利避害，会挑落单的打，但不会设伏——脑子不够用。

这个东西刚才干了什么？它先在3号厂房留了一滩体液，引人进去。自己跑到北面仓库夹道里藏着。等人过去了，撞墙制造声响，让整个包围圈注意力转向北面。然后从东面跑了。

三类干不出这种事。

“往东追！”孙毅在对讲机里喊。

一组的人开始往东移动。姜北带唐遇也在调整位置。

沈擎没动。

他站在右翼的矮墙后面，看着东面。那边是园区围墙，缺口多，出去就是城郊公路。如果这东西真有脑子，它不会往东跑。围墙缺口是最明显的路线，追的人一定往那堵。

它会绕。

沈擎把目光往回收——南面。他刚才过来的那条路。空地，碎石，两边齐腰的草丛。

唐遇的声音在对讲机里突然变了。

不是那种冷调子了。急了。

“信号转向——它没往东——绕回来了——”

停了一秒。

“南面。”

沈擎低头看了看手里马大壮那把磨得发毛的短刀。

草丛在动。

离他大概四十米。不是风。是什么东西在齐腰的草里快速移动，草尖倒伏的方向笔直朝他过来。

三类异种突破了包围圈，绕了一个弧线。

现在正朝唯一没有被包围的方向冲过来。

那个方向只有他一个人。

手里是空的。
