# 第二十九章：这个频道没人回话

站起来的时候左肩嘎了一下。

不是断的声音。是骨头跟骨头挤着磨了一下，酸胀带刺痛，从锁骨沿着脖子根爬到耳朵后面。沈擎用右手按了一下肩头，没按出骨茬，大概率没断。肿了一块，按下去软的，里面在渗血。

补剂还有十支。掏出来一支拧开灌了。嘴里苦的，凉的，入胃之后两三秒热感从胃往四肢走。肩膀那块肿没消，但疼退了一层。

姜北已经站起来了。蹲在路边从废弃面包车底下捡了块东西，一根生锈的铁管，大拇指粗，半米多长。他掂了掂，递给马大壮。

“凑合用。”

马大壮看了看手里半截断棍，又看了看铁管。

“这玩意还没棍子沉。”

“你现在扛不住沉的。”

马大壮张了下嘴，没反驳。接过铁管掂了两下，换了个握法。右手在前左手在后，跟握棍子一样的位置，但手指包不住，管径太细了。

“走。”姜北说。

往东线走的路经过城南那条省道。丁字路口往北三百米，省道跟清河路交叉口。路面上全是裂纹和灰色液体，二类尸体隔几米就有一具，空气里腥的闷的混在一起。路灯灭了大半，亮着的那几盏一圈虫子都没有，异潮期间飞虫比人先跑。

路口立着一块牌子。沈擎以前经过没注意过，今晚路灯正好照着——“清源市裂隙风险等级公示”，底下是一个表格，最后一行手写的日期是上个月。风险等级“III级·注意防范”。III级。公示牌上最高是V级。

今晚至少V。

调度频道一直开着。赵姐的声音断断续续地穿过来，各组汇报、指挥调度、物资请求，十几条线搅在一起。她平时说话絮叨，现在不絮叨了，每句话只有指令和数字，嗓子哑了但语速没掉。

“北线一组，当前位置？”

孙毅的声音从频道里钻出来。远，有底噪，但听得清：“清河路老桥北侧。还在。”

还在。就是还没崩。

“东线二组？”

嗞啦一声。频道杂音冒了两秒。二组组长的声音碎着传过来：“……退到福民巷口了……正面堵不住……有居民没出来——”

后面的被杂音盖了。

赵姐：“二组，重复。有居民？”

“……五楼！有人在五楼！我们过不去——”

频道里安静了一秒。

沈擎低头看路。膝盖那条腿走路没事，快走牵着疼。四个人的速度不快。姜北步幅比平时短了一截，他的腿在撑，每一步落地膝盖都有一个不易察觉的微颤。唐遇在最后面，手提终端抱在胸口，闭着眼走路，声呐覆盖范围在前方铺开。盲走，但不盲。

调度频道切了。切到钱旺的专线。

钱旺的声音传出来了。不是平时开会那个腔调。他在打电话。沈擎听到的是半截对话，钱旺这边的。

“——省局值班室吗？清源分局请求增援支援。异潮规模超出预判。东线防线已被突破，有群众被困。请求——”

对面说了什么听不见。钱旺的声音停了三秒。

“什么叫等待指令？我跟你说现在的情况——”

又停了。

“我不需要你转达！我需要你现在给我调人！东线有十几个老百姓困在楼里——”

停。

这次停的时间长一点。五六秒。

钱旺再开口的时候声音压下来了。不是冷静了。是那种压到极限、再多一个字就要炸的低音。

“你跟我说等待指令。你在省局坐着，夜班值班室里暖气开着吧？我这边有人在流血。你让我等。”

对面又说了什么。

“我等不了。”

电话挂了。啪的一声，硬物砸硬物，不知道摔的话筒还是拍的桌子。

耳机里赵姐的呼吸声冒了一瞬。

三秒。

钱旺的声音重新出来。不打电话了，是对着调度室里的人说话。声音有方向感的变化，他在走动。

“调度台，全频广播。”

赵姐：“全频已开。”

“所有组注意。部署调整。”

他的语速很快，但每个字砸得很实。

“一组，守北线，不动。孙毅，你给我守住。”

频道里孙毅的声音：“收到。”干脆。没有平时那股阴阳劲。

“二组，退到福民巷口固守，等三组到了再处理被困群众。”

二组组长应了。

“三组——”

沈擎在路上走着。耳机里传来自己组的代号。

“三组转向东线支援。姜北，你听到了。”

姜北按了专频：“听到了。”

“城南正面我来接。”

安静了。

赵姐的声音先冒出来——

“局长——”

“叫我去。”

两个字之间没有停顿。赵姐那声“局长”的尾音还没收，钱旺的声音就盖上来了。

叫我去。

不是商量。不是通知。是堵。赵姐后面想说的所有话——“你不能去”“你是指挥”“你受过伤没好利索”“你的气机——”全被堵在嗓子眼。

频道里没人接话。沈擎走着，耳机里调度室的底噪漏出来，风扇转动、键盘敲击、某个屏幕在滴滴响。在这些声音后面，有一个很轻的、金属碰桌面的声音。

钥匙。钱旺在拿钥匙。

“赵姐，调度台你盯着。省局那边再打来，让他们等。”

赵姐没说话。

两秒。

“……收到。”

她的声音里有一种沈擎没听过的东西。不是怕，不是急。像被什么堵了一下，咽了，然后接了。

全频广播关了。调度频道切回常规模式。

走了几步。路面上踩到一只二类的尸体没注意，脚底滑了一下，左肩被牵动，吸了口气。

马大壮在旁边。他也听到了。

“钱局自己去城南？”马大壮压着声音说，“他几阶来着？”

“三阶。”唐遇从后面答的。闭着眼，脚步没偏。“退下来快十年了。”

退下来快十年了。十年前是三阶。十年不上一线，气机量还在，但反应速度、实战判断、身体状态，全退了。他去接姜北的位置，接的是城南正面，接的是丁字路口那个碎了一整片的路面。

这笔账算不出来。

调度台开始轮切频道。异潮期间常规操作，每隔一段时间把所有组过一遍，确认通讯在线。

“一组？”

“在。”孙毅。

“二组？”

“在。”

“三组？”

姜北按了一下：“在。”

“后勤组？”

“在。”

切到下一个频道的时候，沈擎听到了一秒半的空白。

不是杂音。不是底噪。是空的。那个频道上什么声音都没有。干净得不正常，正常通讯即便空闲也有底噪。这一秒半是纯粹的无信号空白。

然后恢复了。底噪回来了。赵姐的声音从那个频道里冒出来——

“这什么破设备，切一下断一下。”

嘟囔了一句。翻了下频道表，又嘀咕了一声：“周鹤的频道也没信号……算了，省局那边自己有通讯系统。”

没当回事。切到了下一个频道。

看了一眼姜北。姜北在走路，没看他。不知道听没听到那一秒半。

刚才切的是一组的频道。断了一下。

也许真是设备问题。异潮期间信号干扰大，通讯不稳定不稀奇。

也许。

---

拐过省道弯道的时候东线方向的声音远远传过来了。嘶吼，密的，闷的，跟城南那边像，但方向不同。城南的从正南来，现在这个从东偏南。

唐遇睁眼了。

他停下来。两只手从终端上抬起来，十根手指微张，掌心朝前。声呐切方向。之前覆盖面一直朝着前方探路，现在整个感知锥面往东线方向偏了。

三秒。

“东线方向。距离一千二到一千五之间。二类为主，三类信号四个。居民楼方向集中。”

他报完了。语速跟平时一样。鼻音重，嘴角那道干血裂了一点，有新的血珠冒出来。

然后他做了一个以前没有过的动作。

右手从终端上收回来，握了一下终端的边框。五根手指收紧，指节发白，握了大概一秒。然后松开了。手放回原来的位置，掌心朝前，手指微张。

跟没发生过一样。

沈擎在他旁边两步远。看到了。

没问。

姜北的步子加快了。膝盖的微颤还在，但他在赶路。

“快一点。”

四个人加速。沈擎的膝盖跑起来牵着疼，左肩每一步都在颠，补剂压住的那层痛感隐隐往外拱。马大壮攥着那根铁管跑在最前面，铁管磕在大腿上叮叮响。

耳机里二组组长的声音又碎着传过来了。

“……三组到哪了？快点……五楼那些人还在——”

沈擎跑着。右手虎口破的那只攥着短刀，刀柄上灰色液体干了一半，粘在掌心和刀柄之间。左手空着，摆臂的时候肩膀那块肿一颤一颤的。

回响空仓。补剂九支。短刀多处缺口。左肩伤了。膝盖磕了。

跑向下一个战场。

【第二十九章回顾】
梗概：三组接令转向东线支援。行进途中通过调度频道听到全局态势：北线孙毅一组硬撑，东线二组防线退至福民巷口，有居民被困五楼。钱旺呼叫省局增援被回复“等待指令”，电话中压着火质问值班室。挂断后全频广播部署调整：一组守北线、二组固守等三组、三组转向东线、城南正面他亲自去接。赵姐试图阻止被“叫我去”两个字堵回去。频道切换检查时一组频道出现一秒半纯空白后恢复，赵姐翻频道表还发现周鹤频道也没信号，均当设备故障没在意。三组加速赶往东线。唐遇声呐切向东线方向报数后，右手握了一下终端边框又松开——以前没有过的动作。沈擎看到了没问。
伏笔：①钱旺立场——重大推进。从打电话求援到挂断后亲自上前线，“叫我去”两个字立住了。他不只是老油条，关键时刻拿命填。②一组通讯闪断——为第39章唐遇发现通讯中断异常做铺垫。一秒半的纯空白（非杂音非底噪），赵姐当设备故障。沈擎注意到了但没深想。③周鹤频道无信号——赵姐翻频道表发现，“算了省局自己有通讯系统”带过。为第36章赵姐告知“调资料”时读者回忆“这人异潮的时候在干嘛”做铺垫。④唐遇预震②——握终端。从第25章“都往这边来了”到本章握终端又松开，情感递进的第二步。⑤省局不支援——“等待指令”。上级系统的迟钝与前线的紧迫形成对比。⑥裂隙风险等级公示牌——城南路口出现，III级“注意防范”，沈擎心里“今晚至少V”。⑦钱旺十年未上一线——唐遇提供信息，三阶退下来快十年了。他去接城南正面意味着什么不需要说。⑧回响空仓持续——补剂减至九支。
状态变化：三组从城南转向东线行进中。钱旺亲自去接城南正面。省局增援状态为“等待指令”。补剂从十支减至九支（沈擎途中用了一支）。马大壮武器换为铁管（临时替代）。一组通讯曾短暂中断。周鹤频道无信号（赵姐未在意）。全局部署调整完成。
下章衔接：第30章“东线”——三组赶到东线。居民楼附近大量异种。二组已退到外围。姜北接手指挥。沈擎没等命令自己先绕楼看了一圈，带回三个信息（消防通道、窗户破损楼层、空心砖隔墙）。唐遇预震③：报楼内数据时语速慢了半拍，“十七个”重复了一遍。钱旺转达省局命令——“放弃固守，收缩防线，等待支援。”
