# 第二章：迎检这天我搬了十二趟桌子

沈擎是被马大壮的闹钟吵醒的。

六点半。闹钟是手机外放，铃声是某首DJ舞曲，放在脸盆里。据马大壮说这样更响。整个宿舍跟炸了一样。

“起起起起起！”马大壮光着膀子跳下床，“八点开会，编外七点到，搬桌子。”

“正装呢？”沈擎闭着眼问。

“你那件不是废了？我的你穿不了，我比你宽两号。老唐的更别想——”

最后是赵姐解决的。二楼柜子里翻出一件灰色西装外套，袖子长一截，领子泛黄，后背有一道折痕。

“谁的？”

“柜子里放了两年没人认领。”赵姐把他袖口往上撸了一截，“凑合。别举手就行。”

七点十分，四楼会议室。

这间屋子平时当杂物间，堆着去年的宣传展板、坏掉的监测设备、半箱过期雨衣。全得清出去。桌椅摆U型，桌面擦干净，矿泉水放好，每个位置一份材料，《清源分局2024年度工作汇报》，四十二页，铜版纸封面压膜。沈擎翻了一眼，三类异种处置次数写六十七次。他记得实际不到四十次。

干活的全是编外。一楼大厅，三个正式编制的在抽烟，说是“等安排”。

搬完桌子搬椅子，搬完椅子挪展板，挪完展板扫厕所。赵姐递给他一瓶除垢剂，“别多挤，一瓶三十块，局里就批了两瓶。”

“楼下那仨不搬？”沈擎往一楼方向扬了下巴。

“人家等会儿要站岗迎接，保持精神面貌。”

“行。”沈擎拎起拖把，“我保持厕所面貌。”

赵姐瞪了他一眼，没接。

厕所刷到一半，唐遇进来了。没打招呼，进隔间，出来洗手。洗得很仔细，指甲缝都搓了。

瘦。黑眼圈比沈擎重。西装扣子扣到最上面那颗。

他往外走，到门口顿了一下。

“桌子少一张。”

“啥？”

“二十三把椅子，二十二张桌子。”

说完走了。

沈擎举着拖把愣了两秒。他什么时候数的？下楼一看，真少一张。仓库翻半天找出来，一条腿歪了，塞了个纸团才稳住。

第十一趟。

七点五十，会议室。

前三排是组长和正式编制。后两排编外。沈擎靠窗最后一个位置，唐遇在他左边，眼睛半闭，胳膊交叉。马大壮在另一头，块头大，占了快两个人的宽度。

姜北第二排，后脑勺看不出什么。脖子上贴了一小块创可贴，不知道什么时候的。

孙毅坐第一排正中，腰板笔直，笔帽拧开搁桌上，材料翻到第一页。

八点整。钱旺进来，后面跟三个人。

钱旺今天黑西装白衬衫，头发往后梳了，连皮鞋都反光。跟他平时烟不离手、走路驼背的样子判若两人。但眼袋遮不住，嘴角绷着。

“同志们，首先代表清源分局全体干部职工，对省局督查处检查组莅临指导表示热烈欢迎——”

沈擎走神了。偷偷看唐遇，唐遇不知道什么时候睁开了眼，在盯着桌上材料的某一页。

“——在上级的正确领导下，分局全年共处置各类异种活动四百一十七起，同比增长百分之十二点三，辖区群众安全满意度持续走高——”

五分钟。最后一句是“下面有请省局督查处周处长讲话”。

三个人里中间那个站起来了。

年轻。三十出头，偏瘦，金属细框眼镜，西装合身但不新，衬衫最上面那颗扣子没扣。不像省局干部，倒像是来实习的——如果不看他站在那儿时其他人的反应的话。

周鹤没拿材料。扫了一圈会议室。

沈擎见过修表师傅拆手表——零件一个个镊出来搁绒布上，手稳，不急，每个零件扫两眼就知道成色。周鹤看人就这种感觉。

“不耽误大家时间。”声音不大，会议室一下子安静了，“这次下来不是找问题，是了解情况。最近全省各分局异种活动数据都有波动，清源不是个例。督查处需要一线的真实信息。”

他拿起桌上那份四十二页的材料，翻了翻，放下了。

“做得挺漂亮。”

前排有人笑了。钱旺的微笑纹丝没动。

不到四分钟，讲完了。

散会。人往外涌。

唐遇站起来拍了拍裤子。

“废话真多。”

“谁的？”

“都。”

说完走了。

沈擎挤出来，马大壮在楼梯口等他。

“那个周处长，我跟你说，三十出头的四阶——要么天赋怪，要么有背景，要么两个都是。”

“小点声。”

“我够小声了！”

赵姐从楼上探头：“会议室桌椅归位！检查组不在会议室了，要去各组值班室看。矿泉水没开封的收回来——别扔，下次用。十分钟！”

沈擎看马大壮。马大壮伸出手指头。

“十二了。”

搬完了。沈擎把最后一张桌子推回原位，出了一身汗，借来的西装后背洇了一大片。他拿袖子擦了一下额头，走出来的时候走廊里已经能听见脚步声。检查组从楼下上来了。

赵姐冲过来，一把拽住他胳膊往楼梯口拉。

“快走快走，回宿舍。”

“我材料还没——”

“回去！”赵姐压着嗓子，脸上是那种又急又不好意思的表情，“别让检查组在走廊碰见你们。钱局交代的——编外在场不太好看。快走，他们上来了。”

走廊那头有人在说话了，声音拐过弯就到。

沈擎把手里的抹布塞给赵姐，转身走了。没等电梯，走楼梯，脚步不快不慢。马大壮跟在后面，脸涨红了，嘴唇动了两下，硬是没说话。难得。

经过三楼，一组值班室门开着。上午在楼下抽烟的那几位已经坐好了，衬衫扎得板正，茶泡上了，其中一个正翻材料，桌上矿泉水还是沈擎十分钟前摆上去的。

三楼拐角，他差点跟一个人撞上。

周鹤。

两个人隔了半步。周鹤手里端着个纸杯，热气还在冒。他大概是从调度室那边过来的，走的跟检查组其他人不是一条路线。

沈擎侧身让。周鹤也侧身——往同一个方向。两人又面对面了。

周鹤笑了一下，自己往另一边让了。

“刚才大会上坐最后排的？”

“嗯。”

周鹤看了一眼他身上那件袖口撸上去、后背洇湿的旧西装。目光往下落了一截，停在他手上。指缝里还有刚才擦桌子留下的灰色印子。

“辛苦。”

语气很随意，像路过跟保洁阿姨说的。然后上楼去了。

后面的马大壮小声问了句“那是不是周处长？”沈擎没答，继续走。

回宿舍。

沈擎把借来的西装脱了，搭在椅背上。袖口那截被他撸上去又滑下来过很多次，衬布起了毛。镜子里那件西装松松垮垮挂着，领子泛黄，后背洇了一大片汗。

不太好看。

马大壮换了拖鞋躺在床上，盯着天花板。安静了大概二十秒，在他的记录里已经很久了。

“我跟你说。干活的时候叫我们，来人了往回撵，上午十二趟桌子，搬的是我们，擦的是我们，扫厕所的还是我们，人家站在楼下一根烟一根烟——”

“睡会儿吧。”

“我睡不着！”马大壮翻了个身，弹簧床嘎吱响，“沈擎你就不气？”

沈擎把鞋踢掉，躺下了。

“气。”他说。“气完了。”

马大壮瞪着他，张了张嘴，又闭上了。屋里安静下来。走廊上偶尔有脚步声经过，都是往楼上去的。

沈擎刚闭上眼，裤兜里的手机震了。

不是他一个人。隔壁屋、走廊里，好几个手机几乎同时响。那种嗡嗡声连成一片的感觉，在镇安局的宿舍楼里只有一个意思。

全局群。调度。

**【三级警报】城西工业园区域监测异常，疑似异种活动信号，等级待确认。各组待命。**

马大壮从床上弹起来，拖鞋都没穿。

三十秒后赵姐的声音从走廊那头传过来，中气十足——

“三组！沈擎马大壮唐遇！下来集合！”

沈擎蹬上鞋，抓了件防护背心，挂在门后的，编外发的旧款，拉链头断了一个。出门的时候路过窗户瞟了一眼楼下，院子里那两辆省局的黑别克还停着，司机靠在车门上抽烟。

一楼大厅。姜北已经到了，防护背心扣好，手里拿着对讲机在听什么。唐遇不知从哪冒出来的，也站在那了，脸上还是什么表情都没有。

走廊那头，周鹤从楼上下来了。手里还端着那个纸杯，大概凉了。两个随员跟在后面，其中一个正贴在他耳边说话。

周鹤抬手让随员停了。

看着集合中的三组。歪了一下头，像想起一件有意思的事。

“正好。”他把纸杯搁在调度室窗台上，“我想看看你们的实战处置流程。”

沈擎拉了拉防护背心的肩带。旧款，偏松，晃荡。

手里是空的。
