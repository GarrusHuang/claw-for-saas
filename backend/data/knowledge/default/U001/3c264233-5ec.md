# 第七章：一类清扫补贴为零但必须出勤

三组的面包车还是没修。

左后轮轴承的事报上去快两个月了，采购计划排到下季度。姜北去后勤磨了一圈，借了二组的五菱宏光，二组今天休整，车空着。

马大壮第一个钻进副驾，把座椅往后推到底，膝盖还是顶着手套箱。

“二组这车比咱那个还小。”

“你瘦点不就行了。”沈擎拉开后排车门。

唐遇已经坐在后排靠窗的位置，闭着眼，手里攥着一个塑料袋。

沈擎瞥了他一下。“又晕？还没开呢。”

唐遇没睁眼。“预防性措施。”

姜北上了驾驶座，调后视镜，发动。车抖了两下才着。

“今天跑南郊片区，十二个监测桩点检，顺路把巡查记录补了。”姜北说，“路上看见一类顺手清，别专门找。”

“一类清扫有补贴吗？”马大壮问。

“没有。”

“那为什么要清？”

“因为巡查记录上要写。”姜北把车倒出停车位，“写了清扫零只，月底考核扣分。”

马大壮转头看了沈擎一眼，沈擎正在系安全带，后排左边那根安全带卡扣松了，得用力摁两下才扣上。

“听见了？”马大壮说，“免费干活。”

“听见了。”沈擎扣上了，“第一天知道？”

车驶出分局大门，往南郊方向开。十月的清源天高云薄，路两边的行道树叶子开始发黄。经过一个路口的时候，交通信号灯旁边竖着一块蓝底白字的牌子——“裂隙安全距离提示：本区域距最近裂隙78公里，风险等级：低”。

这种牌子满大街都是。跟消防通道指示牌一样，没人看。

马大壮从兜里掏出两个茶叶蛋，递了一个给姜北。姜北摇头。他又递给后排，沈擎接了，唐遇闭着眼摆手——车刚过一个减速带，他脸色白了一度。

“唐遇你到底什么体质。”马大壮剥蛋壳，碎片掉了一裤子，“觉醒者不是身体素质比普通人强？”

“强的是气机循环系统。”唐遇的声音闷闷的，“前庭器官跟常人一样。”

“那你咋不用气机稳一下？”

“用了。没用。”

沈擎咬了口茶叶蛋，蛋黄有点噎。“马哥，你以后问他话能不能等下车再问。他一张嘴就想吐。”

嘿嘿笑了两声，马大壮不说了。

车开了大概四十分钟，进了南郊。这片区离市区有段距离，工厂多，人少，监测桩的密度比市区高。路边开始出现灰色的金属桩，两米多高，顶上有个圆盘状的传感器，底座带编号牌。

姜北在第一个监测桩旁边停了车。

四个人下来。唐遇下车的时候扶着车门缓了几秒，脸色才恢复正常。

点检流程很简单：看桩体有没有物理损坏，读一下面板上的实时数据，拍照，填巡查表。一个桩五分钟。

沈擎蹲在桩前面看数据面板。绿色的小屏幕，数字跳动——裂隙活跃度指数0.03，波动幅度0.01以下，正常范围。

“这个正常。”他站起来，拍了张照片。

马大壮在旁边拿着巡查表填。“桩体完好、数据正常、周边环境无异常……这表格跟上个月一模一样，能不能直接复印？”

“不能。日期对不上。”姜北说。

他们沿着南郊的路一个桩一个桩地跑。第二个、第三个、第四个，全部正常。到第五个桩的时候，桩旁边的草丛里有一窝一类异种——蜷在排水沟盖板下面，灰白色，跟大老鼠差不多，三只。

一类不需要什么战斗流程。马大壮一人一脚，踩实了，完事。

沈擎在旁边看着，把三只的位置在巡查表上标了。

“这玩意儿最近多了。”马大壮甩了甩鞋底的灰白色黏液，“上个月巡南郊碰见过一窝，两只。”

“数据上确实在涨。”唐遇说。他蹲在排水沟盖板边上看了看，“这个沟连着工厂区的雨水管网。一类喜欢阴暗潮湿的地方，管网里可能还有。”

“那得报上去让人来清管道。”姜北说。

“报了也排不上。”沈擎把巡查表翻了一页，“上季度报过两次了，排在环境处置计划第三批。”

“第三批什么时候？”

“没写。”

继续走。第六个、第七个、第八个，正常。路上又清了一窝一类，两只，在废弃厂房墙根。

到第九个桩的时候，沈擎蹲下来看数据面板，手停了一下。

裂隙活跃度指数0.03——跟前面几个一样。但波动幅度显示0.04。

前八个桩的波动幅度全在0.01以下。

沈擎盯着那个数字看了几秒。0.04。不高，远没到报警阈值0.5。但在一排0.01之间突然冒出来一个0.04，就像一份全是合格的体检报告里忽然有一项偏高。

单独看不算什么。放在一起看，有点扎眼。

他掏出手机拍了张照片。面板、编号牌、周围环境，拍了三张。

“怎么了？”姜北走过来。

“波动幅度高了一点。”沈擎指了一下屏幕，“0.04。前面的都是0.01以下。”

姜北凑过来瞅了一下。“到报警线了吗？”

“没有。差得远。”

姜北没说话，看了几秒。“记上。”

沈擎在巡查表备注栏写了一行字：九号桩，波动幅度0.04，偏离前序桩均值。然后把手机揣回兜里。

第十个、十一个、十二个，全部正常。

十二个桩跑完，快下午两点了。四个人站在最后一个桩旁边，马大壮的肚子叫了。

“回去吃饭？”

“路上有家拉面。”姜北说。

“行。谁请？”

没人接话。

“AA。”姜北说。

马大壮上车前朝南边望了一眼。那个方向是裂隙所在。天际线上什么也看不见，晴天的时候裂隙的光被太阳盖住了。但知道它在那儿，就像知道远处有座看不见的山。

---

拉面馆很小，路边店，四张桌子。老板认识镇安局的制服，面多给了半勺。

马大壮要了大碗加肉，姜北中碗，沈擎中碗加蛋，唐遇要了碗清汤的——刚坐下脸色才彻底缓过来。

“唐遇你说这车再开半小时你会怎样？”马大壮问。

“你不想知道。”

沈擎把蛋搅进面汤里。“马哥你别老逗他，上次他吐你鞋上你忘了？”

吸了口面，马大壮含糊说了句什么，听不清。

面馆的电视挂在墙角，放着本地新闻。画面里是一段裂隙的远景航拍，配着播音腔的旁白：“……镇安局表示，本市裂隙活跃指数持续处于可控范围，市民无需恐慌……”

四个人都没看电视。这种新闻跟天气预报一样，永远是“可控”。

唐遇慢慢喝着汤，突然说了句：“你们说七类要是真有，长什么样？”

“七类？”马大壮抬头。

“培训教材最后一页。理论推导出来的等级。”唐遇说，“一到六类都有实际记录，七类没有。'理论上存在'。”

“理论上存在就是还没出现呗。”马大壮说，“别出现最好。六类就够呛了，省局去年处置那个用了多少人？”

“一个大队加两个中队增援。”唐遇说。

“那七类得全省动员了吧。”

“教材上写的是'全省动员级'。”唐遇喝了口汤，“但那是理论，真来了什么样谁也不知道。”

“不会来的。”马大壮说，语气很随意，像在说“明天不会下雨”。

沈擎没参与这个话题。他在看窗外——拉面馆门口的马路对面，隔着一排行道树，能看见远处一根监测桩的轮廓。两米多高的灰色金属杆，顶上的圆盘在太阳底下反着光。

“我老家那边监测桩多得很。”沈擎说，像是自言自语，“小时候当路标用。出门跟人说往第几根桩那边拐。”

“临渊镇？那地方桩多正常，离裂隙近嘛。”马大壮嘿了一声。

“嗯。”

姜北一直在低头吃面，这会儿也没抬头。

沈擎目光落在他身上。

姜北把最后一口面汤喝了，放下碗，擦了擦嘴。“走吧。”

话题就这么过去了。

没人接。也没人追问。

但沈擎注意到姜北放碗的时候手稳得很——稳得像刻意的那种稳。

---

回分局的路上，唐遇又闭上了眼，手里攥着塑料袋。马大壮打了个饱嗝，靠着车窗开始眯。

沈擎坐在后排，翻手机里拍的照片。

九号桩。0.04。

他往前翻了翻之前巡查的记录，上个月的、上上个月的。九号桩以前的波动幅度是0.008、0.009、0.011。一直在0.01附近。

0.04。涨了四倍。

绝对值不高。但趋势不对。

他把手机锁屏，揣回兜里。

车窗外，南郊的工厂和空地往后退。远处天际线上干干净净的，什么也没有。
