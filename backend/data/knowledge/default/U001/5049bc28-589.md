# 第二十三章：第一声警报响的时候我在上厕所

战备第十一天。下午两点四十。

沈擎蹲在二楼厕所的隔间里。

老式蹲坑，头顶日光灯管坏了一根，另一根隔几秒闪一下。

他在看手机。唐遇十分钟前发了条消息：“南线十四号桩读数0.31。”

0.31。报警阈值0.5，但两天前十四号桩还是0.09。两天涨了三倍多。

他把消息划掉，退出来，看到手机右上角信号只有一格。城南这边信号一直差，靠近裂隙的区域都这样。

正想站起来。

警报响了。

不是平时的三级蜂鸣，是一条直线拉过去不停的长音。一级。

整栋楼在震。楼顶警报器功率太大，铁皮共振，日光灯管跟着抖，坏的那根掉下来了，砸在地上碎了一片。

沈擎站起来的时候裤子还没提好。一只手拽裤腰一只手推门，冲出隔间。走廊上已经有人在跑，很多人同时在跑。有人在喊什么，被警报盖住了。

跑回值班室。门开着。马大壮已经在穿防护服了，裤腿只套了一条，另一条腿踩在地上蹦。唐遇站在终端前面，屏幕上的数字在跳，红色的，很多红色。

“几级？”沈擎扯开柜子拿防护服。

“一级。”唐遇的声音跟平时一样平，但语速快了一倍，“全市监测数据十分钟内飙了三倍。南线所有桩全红。”

全红。十七根桩，全红。

姜北不在。

“姜哥呢？”

“调度室。”唐遇头也没回，手指在终端上划，“钱局叫的。”

防护服套上了。新款，深灰色，姜北给的那件。拉链拉到领口的时候手指顿了一下，拉链头是冷的。

走廊上的脚步声越来越密。有人跑过值班室门口，防护服只穿了上半身，手里攥着头盔往楼梯口冲。

把急救包扣在腰后，通讯耳机塞进左耳。频道里全是杂音，好几个人在同时说话，分不清谁是谁。

他拧了一下频率，切到三组专频。

安静了。只有底噪。

马大壮终于把两条裤腿都套进去了。他从柜子里抄起那根缠了新胶带的棍子，胶带缠得不匀，有一截鼓出来。

“走。”马大壮说。

“等姜哥。”

话音没落走廊上传来脚步声。快，重，节奏稳。姜北。

他出现在门口的时候已经穿好了。旧款防护服，灰黄色，就是跟沈擎换的那件。左手提着头盔，右手拿着一张纸。

“三组，城南防线。”他把纸往桌上一拍，没停步，“车在楼下，两分钟。”

纸上是手写的，钱旺的字，歪歪扭扭，像是站着写的。沈擎扫了一眼：三组/城南/丁字路口/南郊路×工业三路东北五十米。

丁字路口。他们讨论的那个位置。钱旺批了。

唐遇把终端锁了，拔了U盘揣兜里。手提终端夹在胳膊底下，另一只手拿着耳机线边走边往脖子上绕。

四个人出了值班室。走廊上挤满了人，各组都在往楼下跑。有人背着装备箱，有人拎着补剂盒，二楼拐角处两个人撞在一起，装备箱摔开了，零件撒了一地，蹲在那儿捡。

没人管。没时间。

经过调度室门口的时候沈擎往里扫了一眼。门半开。

钱旺站在调度台后面。面前四块屏幕全亮，三块是地图，一块是通讯频道列表。他左手撑着台面，右手拿着对讲机在说话。说什么听不清，被走廊的脚步声盖了。

一眼扫到了钱旺的右手。在抖。对讲机在手里微微震动，不是按键在震，是手本身在抖。但他声音从对讲机里传出来时是稳的。

赵姐坐在调度台侧面，面前摆了三部座机和一沓签收单。左手一部右手一部，嘴里跟左边那部说着什么，右手那部夹在肩膀和耳朵之间。

一秒。收回目光，跟上姜北。

---

楼下停了三辆面包车。引擎已经发动了。三组的车排最后，白色金杯。

姜北拉开驾驶座的门坐进去。马大壮副驾驶。沈擎和唐遇后排。

后排座位上扔着两箱东西，补剂备品和急救耗材。沈擎把补剂箱挪到脚边，坐下。

车门关上。姜北挂挡。前面两辆已经出了院子，出大门往南拐。

省道上没什么车。平时这个点南郊公路上偶尔有货车，今天一辆没有。路两边的店面也关着，卷帘门全拉了。

该是有人提前通知了疏散。或者他们自己跑的。住在城南的都知道，警报响了往北跑。

车窗没关严，风灌进来。南风。渊味比那天夜里在窗口闻到的浓了不止一个档，呛人的，烧焦铁锈味钻进鼻腔往嗓子里走。

唐遇在后排打开了手提终端。屏幕亮了，他划了两下，调出实时监测页面。

“南线全桩过阈值。”他说，语速快但一个字不含糊，“最高的是九号桩，0.87。十一号、十四号也过0.5了。省局的数据显示裂隙涌出量……”他顿了一下，盯着屏幕上的数字，“从十分钟前开始，每分钟涌出个体数量在两百以上。二类为主，目测混有三类。”

每分钟两百只以上。

“前面有车。”马大壮说。

省道前方两百米，一辆面包车停在路边，双闪亮着。一组的车。车门开着，没人，他们已经到位了。一组的防区在城南偏北，比三组靠后。

没减速，从旁边过去了。

路在往南走。两边的房子越来越矮，越来越旧，有些门窗已经用木板封了。南郊迁走了大半居民，剩下的都是走不了或不想走的。

他看了一眼右侧车窗外。

下午两点多，太阳应该还高。但南边天色不对。偏暗，像把亮度调低了两档。地平线那个位置是亮的。那条光不再是夜里才看得见的细线了，大白天它还是亮的，沿着地平线铺开，灰橙色，比天亮。

“姜哥。”沈擎说，“丁字路口，按说好的来？”

姜北从后视镜里看了他一眼。

“按说好的来。”

四个字。车在颠。省道过了收费站之后路况差了，坑坑洼洼，底盘磕了两下。马大壮一手撑着车顶，另一只手攥着棍子。

唐遇的终端屏幕上红点在增加。一片一片地冒出来。

“三公里了。”唐遇说，“我能扫到了。”

他闭上眼。手指微张，手提终端搁在膝盖上没再看。

两秒。

“正南方向，二类群，数量……”他顿了一下。

“多少？”

“没数完。持续涌出中。”唐遇睁开眼，看着姜北的后脑勺，“三类混在里面，我扫到至少四只。可能更多，后面的信号叠在一起分不开。”

车里安静了一会儿。

引擎声，风声，底盘磕在坑上的闷响。

“到了减速。”姜北说，“别冲过头。”

前方路口出现了。丁字路口。南郊路在这里分岔，往西拐进那条窄街，三米多宽，两边是废弃的平房。北边那条路直通收费站方向。

车停在路口北侧二十米处。拉手刹。熄火。

四个人下车。

风更大了。南风裹着渊味扑面过来，沈擎眯了一下眼。防护服领口的反光条在风里拍着脖子。

南边的路空荡荡。省道直直通向裂隙方向。远处地平线上那道浑浊的光比在窗口看近了很多。肉眼可见的距离里还什么都没有，但唐遇说三公里外已经扫到了。

三公里。二类的移动速度，十分钟左右。

姜北已经走到丁字路口正中站定了。头盔扣上，系带拉紧。两脚分开与肩同宽，踩在柏油路面上。就是站在那儿了。

马大壮拎着棍子往西侧窄街走。走到巷口停住，回头看了沈擎一眼。咧了一下嘴，不知道是笑还是咬牙。

沈擎站在姜北右后方五米。机动位。

他摸了一下腰间的短刀。刀柄上的胶皮磨得发亮，是旧的，自己买的，报销没批过。

手里是空的。回响，空仓。一个半月了。

补剂在腰包里，防护服是新的，短刀在手边，唐遇的声呐开着，姜北站在前面，马大壮守着侧翼，退路在北边。该有的都有了。除了回响。

耳机里传来唐遇的声音。平，稳，快。

“两公里。正面方向。前锋全是二类，数量超过五十，还在增加。三类信号四……不，五个。后面的还在涌。”

风把渊味往脸上糊。沈擎没眯眼。临渊镇的人不怕这个味道。闻着它长大的。

远处的省道尽头，地面上出现了移动的东西。灰褐色，矮矮的，贴着地面。一只、两只、五只、十几只，看不到头。像潮水从低处往这边漫过来。

沿着路面，沿着路两边的排水沟。

水往低处流。

“一公里。”唐遇说。

姜北的脚碾了一下地面。柏油路面发出一声闷响，裂开了一条细纹。

他没回头。

“三组。”姜北说。

沈擎握紧了短刀。

【第二十三章回顾】
梗概：战备第十一天下午，一级警报突然响起——全市监测数据十分钟飙三倍，南线全桩过阈值。全局出动。沈擎从厕所冲回值班室，四人穿装备集合。经过调度室看到钱旺拿对讲机的手在抖，赵姐同时接两部电话。三组驱车赶往城南防线，路上唐遇实时报数据：裂隙每分钟涌出个体两百以上，正面二类超过五十只且持续增加，三类至少五只。到达丁字路口，四人按第22章讨论的部署就位——姜北正面、马大壮侧翼窄街、唐遇北侧声呐、沈擎机动。远处省道尽头出现大量灰褐色移动体，沿路面和排水沟涌来。沈擎确认：水往低处流。
伏笔：①战术衔接——兑现。丁字路口卡位被钱旺批准（手写调令纸上写了具体位置），第22章的讨论从纸面落到了地上。②钱旺——保温。手在抖但声音稳，一个画面带过。③赵姐——保温。调度台侧面同时接两部电话，功能性出场但有画面感。④回响空仓——保温。“一个半月了”。⑤裂隙视觉升级——推进。下午日光下裂隙的光仍可见，从夜间细线→白天肉眼可见。⑥渊味——推进。从窗口若有若无→车里呛人。⑦“水往低处流”——呼应第22章判断，异种确实沿低洼地势涌来。
状态变化：全局一级警报，异潮正式爆发。三组到达城南防线丁字路口就位。回响仍空仓。沈擎带着短刀、补剂、新款防护服上阵。
