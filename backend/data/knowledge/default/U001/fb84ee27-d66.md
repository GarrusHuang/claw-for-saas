# 第十五章：天台上的谈话

天台的门没锁。

这扇门从沈擎来分局第一天就没锁过。锁芯坏了，报了两次修，没人来。后来赵姐在门把手上缠了一圈胶带，算是提醒“非必要别上去”。

姜北推开门，胶带扯断了一截。他没管。

天台不大。水泥地面，几根通风管道，一个锈了的太阳能热水器架子，早不用了。靠北边有半米高的矮墙，矮墙上落了一层灰。

姜北走到矮墙边上，没坐，站着。

沈擎跟出来，把门虚掩上。门框变形了，关不严实，风从缝里挤进来，发出细小的哨声。

风比楼下大。九月底的夜风，不冷，但吹在后脖颈上有点凉。远处城南方向天际线上，裂隙微光隐约可见——那道光今年比去年亮。沈擎还记得小时候在临渊镇，裂隙的光只有月亮的几分之一。现在隔着整个清源市区都能看见。

姜北没马上说话。他摸出一包烟，抽了一根点上，吸了一口。

沈擎靠在通风管道边上，等着。

烟抽了小半根，姜北才开口。

“你那个本事。”

沈擎没接。等他说完。

“局里的体检报告上写的是D级。”姜北看着远处，“复制型，实战价值有限。”

“嗯。”

“入职体检做的？”

“对。体检中心那边安排的测试官，二阶，出了三招。我录了一个，打出来跟他差不多。评估的人写了个D级。”

姜北把烟灰弹了一下。“二阶测试官，出的是教学用的标准攻击。你录了一个标准攻击，打出来跟原版差不多——C级复制二阶，还原度高但上限低。给D级不算冤。”

“不冤。”

沈擎说得很平。

他确实没觉得冤。体检那天，那个测试官出招跟练体操似的——动作标准，没脾气，一板一眼。沈擎录了，打了，评分的人看了一眼，在表格上写了个D。

没什么好说的。那个环境下，D级就是合理的。

姜北把烟叼在嘴里，转过头看沈擎。

“但昨天你复制的是三阶的。”

沈擎没吭声。

“孙毅那个劈斩，我见过很多次了。他三阶稳了两年多，那一招是他吃饭的本事。”姜北的语气没什么起伏，像在说今天食堂的菜咸了，“你挨了一刀，然后原样打出来。力道是差了很多——但轨迹、发力方式、甚至他左脚的那个小动作。一模一样。”

“嗯。”

“体检的时候你复制二阶，打出来是二阶的。昨天你复制三阶，打出来轨迹是三阶的，力道被你C级的气机拉下来了。”

姜北看着他。

“你这个东西的上限不在你身上。在对面。”

风灌过通风管道，呜了一声。

沈擎没说话。

不是不想说，是这句话他自己琢磨过，但从来没被人这么直接讲出来。

他知道回响就是这样。临渊镇那会儿就隐约知道了。那时候他录过民兵队长一招枪刺——一个退伍老兵，正经练过的。打出来，旁边人愣了半天。后来在镇上又录过几次，对手越强，打出来的东西越吓人。

但没人帮他总结过。十五六岁的小孩不会往这些事上琢磨。后来签了编外，体检给了D级，他就当自己是D级了。D级就D级，反正也不影响上班——影响的是工资和配额，但那是编制的问题，不是D级的问题。

他一直是这么想的。直到刚才姜北那句话。

“所以评估报告上那个D级——”姜北把烟在矮墙上摁灭了，“是因为测试环境太弱。不是你弱。”

沈擎靠在管道上，呼了口气。

“姜哥，你拉我上来就为了说这个？”

姜北把烟头揣进口袋。天台上没垃圾桶，他从来不把烟头丢地上。

“周鹤的人今天在场。”

沈擎的手指动了一下。

“昨天演练的时候，场边有个面生的。”姜北说，“穿咱们分局的训练服，但三组一组二组机动组我都认得。那个人不是分局的。”

沈擎想起来了。散场的时候，人群后面，掏手机的那个。

“我不认识他。”

“我也不认识。”姜北靠在矮墙上，“但周鹤上次来检查带了四个人。有两个没在大会上露面。”

两个没露面的。

沈擎记得。迎检那天大会上，周鹤身后站了两个人，都是省局督查处的。但周鹤是带着一个小组来的——检查组的编制通常是四到六人。

“你的意思是，有人留在清源了。”

姜北没正面回答。他换了个话头：“你昨天那一下，在场的人都看见了。赵姐写了记录。这些是正常的，演练本来就有记录。”

他停了一下。

“但如果有人拍了视频发回省局，那就不一样了。”

沈擎咂了咂嘴。

“不一样在哪？”

“你自己想。”姜北看着他，“一个D级的编外协勤，在内部演练里复制了三阶正式编制的看家招式，当众打出来。这个事，你是分局局长，你会怎么想。”

沈擎没接话。

他知道姜北不是在考他。是在提醒他。

如果周鹤手里有这段视频，那周鹤看到的不是“一个编外临时工耍了个小聪明”。周鹤看到的是——这个人的能力评估可能有误，或者这个人一直在藏。

不管是哪种，省局督查处的人都会很感兴趣。

沈擎把这笔账在脑子里过了一遍。周鹤上次来就问过他临渊镇、问过打法来源。那时候沈擎以为是例行公事。现在加上昨天的视频——如果真有视频的话——周鹤手里已经有两样东西了：一份D级的体检报告，和一段不像D级的实战录像。

这两样放在一起，任何一个做督查的人都不会当没看见。

“我没藏。”沈擎说。

“我知道。”

“体检就是那个结果，不是我故意考低了。”

“我知道。”姜北重复了一遍，“但周鹤不知道。”

天台上安静了一会儿。风把通风管道吹得嗡嗡响，和远处街上的车声混在一起。楼下停车场有人在倒车，倒车雷达滴滴响了好几声，笨手笨脚的。

姜北伸了个懒腰。不像他平时的动作——他很少在别人面前做这种放松的举动。

“以后用的时候注意场合。”他说，“能不亮的时候别亮。”

“那昨天呢？”

“昨天他打你了。”姜北的语气没变，“挨了打不还手，像什么话。”

沈擎愣了一下。

这话从姜北嘴里说出来，不太对味。姜北是那种“忍一忍过去了”的人，他什么时候觉得挨了打应该还手了？

但他没深想。因为姜北说完这句之后，脸上的表情很短暂地松了一下——几乎看不出来，但沈擎站得近，看见了。

“行。”沈擎站直了，“还有别的吗？”

姜北想了一下。“明天早上六点。”

“知道了。”

沈擎转身往门口走。

“沈擎。”

他停下。

姜北背对着他，还靠在矮墙上，看着城南方向那道微光。

“你那个本事，自己心里有数就行。别到处说。也别到处用。”他顿了一下，“但真到用的时候，别省着。”

沈擎推开天台的门。

门把手上的胶带剩半截，耷拉在那儿。

“姜哥。”他回了一句，“我什么时候省过。”

下了楼梯，走廊的灯管还在闪。

值班室门开着，里面传来马大壮的声音——“你说他俩在上面聊什么呢？”

唐遇的声音：“不关你事。”

“我就好奇——”

“不关你事。”

沈擎进了门。

马大壮靠在椅背上，手里攥着手机，屏幕上是外卖软件，购物车里躺着一份卤肉饭。看见沈擎，手机往桌上一扣，嘴巴张开又合上，最后憋出一句：“聊完了？”

“嗯。”

“聊啥了？”

沈擎走到自己位置坐下，拽过桌上一袋唐遇的压缩饼干，掰了一块。

干。硬。没味。像在嚼纸板。

唐遇看了他一眼，没说什么。那包饼干他一般不让别人碰。

“没聊什么。”沈擎说。

【第十五章回顾】
梗概：姜北在天台跟沈擎谈话。核心内容三层：一、点明回响体检报告D级的本质——不是沈擎弱，是测试环境太弱，回响的上限取决于对手而非自身；二、警告周鹤的人可能在场拍了视频，省局督查处对沈擎的关注度将升级；三、交代原则——能不亮别亮，但真到用的时候别省着。姜北罕见地肯定了沈擎昨天还手（“挨了打不还手，像什么话”）。
伏笔：①回响——机制第三次揭示。“上限不在你身上，在对面”被姜北直接点破。D级评估的局限性首次被分析。读者现在完整理解了“越惨越强”的逻辑。②周鹤暗线——重大推进。姜北指出演练场的陌生面孔可能是周鹤留在清源的人，检查组可能有两人未在大会露面。如果视频发回省局，沈擎会被重新评估。③姜北性格——深化。天台上第一次在沈擎面前伸懒腰（放松姿态），用“挨了打不还手像什么话”间接肯定沈擎。烟头揣口袋不丢地上。
状态变化：回响仍空仓。沈擎对自身能力的认知更清晰（上限取决于对手）。对周鹤的警惕提升。姜北与沈擎之间的信任度上升一级（天台私谈=区别对待）。
下章衔接：第16章“监测桩坏了三根”——巡查发现更多异常，之前九号桩附近又有两根波动+一根物理损坏（像被从地下拱断）。三组上报，钱旺反应慎重。夜里沈擎搜到旧新闻——三年前另一个分局也出现过监测桩损坏，两个月后爆发小规模异潮。
