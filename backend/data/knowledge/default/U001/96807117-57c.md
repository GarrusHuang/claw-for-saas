# 第二十七章：我看到了

下一波没给喘气的时间。

唐遇说八百米的时候沈擎还靠在歪电线杆上。唐遇说五百米的时候他已经站回了机动位。中间那几十秒他做了三件事：拧开一管补剂灌了，把短刀在裤腿上蹭了两下，把右手手套摘了，虎口破了的那只，粘在掌心上扯不利索，干脆不戴了。

光手握刀柄。金属冰，贴着生肉。

“三类八只，混在二类群中间。二类五十以上，分不清了。”唐遇的声音平，快，鼻音比之前重。“正面五只三类。左侧排水沟两只。右侧废弃房方向一只。”

八只。之前最多的一波才六只。

正面涌过来了。这一波的声音跟前面不一样，前面几波二类打头，嘶吼密，碎，像一层毛刺盖在路面上。这一波，嘶吼中间有更低的东西在震。三类的声音。从胸腔里挤出来的，闷的。

好几个在同时叫。

姜北站在路中间。脚下已经是一整片碎裂区了，碎块、灰色液体、二类的尸体碎片混在一起，他站在中间一个坑里，比路面矮了快五公分。两脚碾下去，坑底又裂了一层。

第一只三类冲上来。正面。姜北接了。定桩。冲击力沿腿灌下去，坑底碎块弹起来。这只比普通三类大一点，但不多，甲壳粗糙，扁平头。一拳弹出去，它翻了个滚。

第二只紧跟着。没等第一只落地。姜北左肘横扫，第二只侧飞出去，撞在路边废弃面包车上，车门从铰链上脱了下来。

第三只从左侧排水沟里翻出来。

“大壮！左边！”

马大壮从窄街口冲出来。备用棍——短了一截的那根——抡在三类侧面。三类被打偏了方向，撞进路边平房院墙。墙倒了半面。马大壮追进去补了一棍。

右边又来了一只。沈擎迎上去。

不是迎，是挡。C级气机，短刀。三类正面冲过来他接不住，他在等它冲过姜北之后露出侧面的那一瞬。三类从姜北右侧两米处掠过，它的注意力在姜北身上。沈擎从斜后方追了两步，刀捅进侧腹甲缝，勉强穿透，手上传来骨头磨金属的阻力。三类嘶了一声扭身，尾部横扫过来。他松刀跳后半步。尾巴擦着胸口过去，风压把防护服前襟拍了一下。

刀还插在它身上。

三类带着刀转了个圈，甲缝里灰色液体往外冒。它没倒，但慢了。姜北回身一拳补在它头侧。倒了。

沈擎上去拔刀。刀柄滑，灰色液体混着血。拔出来的时候手差点脱。没戴手套的右手，光皮肉贴着金属，热的粘的全传上来。

正面还在涌。五只三类冲进来了三只，姜北扛了两只。第四只从排水沟里钻出来顶翻了盖板，铁盖板飞起来砸在唐遇前方三米。唐遇没动。这已经是今晚第二次了。

第五只绕了个远路，从左边废弃平房的巷子里窜出来——窄街方向。

“窄街有！”沈擎喊。

马大壮还在院墙废墟里跟第三只纠缠。来不及。

姜北转身。面前两只三类还没处理完。他打了一拳把左边那只弹开，右边那只趁这一瞬扑到了他肩上。甲壳压着他往下沉。单膝几乎触地，又撑起来了，把它从肩上掀翻出去。

但窄街那只已经冲过去了。往北。往撤退通道方向。

“放了一只。”沈擎在频道里报。

“二组会接。”唐遇。

频道里二组的声音传过来，模糊，远，听不清说了什么。

这一波的二类还在涌。沈擎杀了三只从姜北侧面溜过来的，第四只扑在他小腿上。新款防护服扛住了咬，但力量把他带倒了。膝盖撞在碎石上，刺痛。从地上捅了一刀，二类软了。爬起来。

膝盖那一下磕得不轻。走路没事，跑会疼。

八只三类清了七只，跑了一只。二类不计其数。打完的时候沈擎的耳朵嗡了一阵。他站在原地等嗡声过去。

唐遇在耳机里说了个数字。下一波。

上一波的尾巴还没扫干净，下一波的前锋已经到了视线里。没有间隙了。

它还在那里。

一百多米外。裂隙的灰橙色光勾着它的轮廓。四条腿，骨板弧面，竖瞳对着丁字路口的方向。

所有东西都从它两侧流过去。二类绕着它跑，三类从它身边经过时步子碎了一下，不自在。没有一只碰它。

沈擎一边处理从侧面漏过来的二类一边看它。

它在等什么？

上一次它冲过一次。姜北接住了。它退回去了。现在又站着不动。

这一波比上一波还猛。三类的数量唐遇已经报不过来了，新的到得比他报的快。姜北正面扛着，拳肘交替。他的呼吸变了——不再是鼻息，是嘴巴也在出气，短促的，每一拳呼一次。

姜北在累。

三阶打三类不难。但打到现在，不知道多少波了。气机在磨。体力在磨。脚下的碎裂区让他站的姿势越来越深，每次蹬地发力都要从一个坑底往上使。

“唐遇，侧翼。”频道里姜北的语速正常，但词和词之间的缝隙宽了一点。

“左侧排水沟三只三类逼近，六十米。右侧窄街方向两只二类，不是三类。后方清。”

三只三类从左侧排水沟来。正面还有两只三类没解决。

扛不过来了。

“大壮。正面来搭把手。”姜北说。

马大壮从窄街口跨出来。备用棍扛肩上。棍中间的裂纹在裂隙光映照下能看见一条暗线，从棍身一侧延伸到另一侧。

他到了正面。一棍抡在三类侧面，三类被打偏了两米。棍子传来的震动让他手臂抖了一下。裂纹在扩。

左侧排水沟那三只到了。盖板被从底下顶开，三只三类依次翻出来。前面两只扑向姜北的左侧。第三只稍慢，往沈擎方向偏了。

沈擎没退。等它过了姜北的位置线，从右后方绕到它转身的盲区。三类转身慢，后肢蹬地有顿挫，那一两秒侧面是空的。刀捅进去，捅得比之前浅——刀刃的缺口挂住了甲缝边缘。拽出来，连皮带肉扯了一条。三类嘶了一声回头要咬，姜北的拳头从正面过来了。

倒了。

正面还在涌。

姜北处理了左侧两只之后面前又来了三只。马大壮在右侧堵着，棍子每一下打出去手都在抖。

“唐遇——后面还有多少？”马大壮喘着喊。

“正面方向后续三类——”唐遇停了一下。“那个大的动了。”

沈擎转头看。

一百多米外。那个轮廓变了。

头低下来了。脊背从水平变成前倾。骨板缝隙张开了一线，微微的，像一台机器在预热。

它在蓄力。

上一次他在一百一十米的距离上看过这个动作。头低，脊背前倾，骨板缝隙打开。然后是全力冲刺。

但上一次他在手忙脚乱地退。这次不一样。他站在原地，没有在挡什么，也没有在追什么。

他在看。

脚趾。它先动的是脚趾。四只脚的脚趾同时抓地，趾尖陷进柏油路面。力从脚趾沿着踝关节往上走。踝关节锁死。

然后是膝。四条腿的膝关节同时微曲，角度几乎一样，像四根弹簧在同步压缩。大腿肌群在骨板覆盖下面鼓胀，从底下把骨板往外撑。

脊椎。从尾部开始弓起来，一节一节往前传。力的波像水一样沿着脊柱走。走到肩胛的时候骨板前端下压，两块肩胛骨之间的缝隙最大限度张开了。

最后是四肢的弹射。同步的。脚趾松开地面的瞬间，所有积蓄的力同时释放。

整个过程不到四秒。

但沈擎把每一帧都看进去了。

他看的是力怎么走。从哪里开始，经过哪里，在哪里转弯，在哪里汇聚。脚趾→踝→膝→大腿→脊椎弓→骨板前倾→肩胛挤压→四肢弹射。一条完整的力的路线，从地面开始，最后全部灌进冲刺的方向。

它冲的是姜北。

一百多米的距离它三秒跑完。地面在震。骨板弧面在裂隙光里拖出一条暗影。

姜北蹬地。两脚碾进碎裂区的坑底——坑底再碎。小腿绷直，膝盖锁住。定桩。

撞上了。

传来的全是碎裂声。整片路面从姜北脚下往外炸开，蛛网状裂纹扩了三米多远。碎块飞起来打在两边废弃面包车底盘上、打在路灯杆上、打在沈擎的防护服上。一块碎石弹到他脸颊，火辣辣的。

姜北的膝盖弯了。

不是一瞬。这次弯了下去，弯了一个很明显的角度，能看到大腿肌肉在发力往回撑。上半身也晃了。之前那种冲击力完全不经过身体的完美状态，这次做不到了。力太大了。定桩能导的量有上限，超过上限，多余的力就会漏上来。

膝盖撑住了。撑回去了。

姜北一拳打出去。

变异三类被弹回去。六七米远。四条腿在路面上刮出又深又长的痕迹。站住了。骨板上多了一道痕，比上次的深一点，但也只是深一点。

马大壮跟上去了。备用棍全力抡下来，砸在变异三类侧面骨板上。

白印。

打普通三类甲壳至少能打出凹痕。这层骨板把力全吃了，什么都没留下，跟指甲刮了一下一样。二阶的力穿不透这层壳。

马大壮退了两步。手指发麻，棍子差点脱手。

它还站着。

姜北也还站着。但他的双腿在抖，很细微，从膝盖往下。脚下的碎裂区已经成了一个凹陷，比路面低了快八九公分，边缘碎石斜插着。

他还能再接几次？

变异三类盯着姜北。那两道竖瞳在裂隙光里亮了一下。它没急着再冲。退了几步。站住。

在看。在算。它也在算。

沈擎攥着短刀。右手没戴手套，掌心和刀柄之间全是汗和灰色液体，粘腻的。

他在算另一笔账。

刚才那四秒。脚趾→踝→膝→大腿→脊椎弓→骨板前倾→肩胛挤压→四肢弹射。一条完整的力的路线。他看到了。每一个节点在什么时候发力、力从哪个方向走、怎么汇到最后那一下冲刺里。

看得清清楚楚。

好碟。

但有个问题。回响的条件——它得冲着他来。有攻击意图，对着他。

刚才那一下，它冲的是姜北。沈擎站在侧面五六米的位置。竖瞳对着姜北，冲刺方向对着姜北，全力输出对着姜北。

不是冲他来的。录不上。

除非再来一次，冲他。但那个东西三米多长，骨板覆盖，全力冲刺连姜北都接到极限。它冲他？他接不住。不是接不好，是会死。

这张碟，看得见，拿不到。

沈擎盯着一百多米外那个轮廓。它又站住了，四条腿踩着，竖瞳亮着，在等下一次蓄力。

它会再冲。再冲的时候还是冲姜北。因为姜北站在正面，姜北是威胁。沈擎不是。

手里空的。

变异三类低下头。骨板缝隙又开始张了。

第二次蓄力。

同样的起手。脚趾抓地。踝关节锁死。膝关节压缩。脊椎从尾部开始弓。

沈擎没动。他站在姜北右侧五米，看着。

力的路线跟第一次一模一样。这次他在确认。第一次看到的每一个节点，对不对。脚趾先抓、踝先锁、膝先压——没有变化。连角度都一样。这是它的本能，不是技巧。本能不会变。

肩胛骨之间的缝隙张到了最大。

弹射。

冲的是姜北。

但这一次，沈擎的整个人抖了一下。

不是因为地面在震。是他体内有什么东西动了。

从胸口往四肢走的。热的。不是气机，他的C级气机他熟，不是这个温度，不是这个走法。这个东西更深，更密，沿着他从来没感觉过的路线在跑。像有人拿针沿着他的骨头往里缝。

回响。

激活了。

他愣了半秒。

它冲的是姜北。不是他。竖瞳对着姜北，冲刺方向对着姜北，全力输出对着姜北。按他摸出来的规则，不冲他来的，录不上。

录上了。

骨头缝里那个热的东西还在走。从胸口到肩，到肘，到腕，到指尖。同时往下走，腰，胯，膝，踝。一条完整的路线在他体内铺开——脚趾→踝→膝→大腿→脊椎→肩胛→四肢。力的路线。不是他自己的力。是它的。

变异三类撞上姜北的那一瞬间，沈擎什么都没做，站在旁边五米。

但他的身体在记。

变异三类被弹回去。姜北的膝盖又弯了，这次弯得更深，右脚脚跟从地面抬起来了一瞬——地面碎到兜不住了。他硬撑着蹬回去。

沈擎没看姜北。

他低头看自己的手。右手，光着的那只。手指在微微痉挛。回响在走路线的末梢反应。

手里有东西了。

攥了攥拳。松开。攥了攥。

热的。涨的。塞得满满的，跟以前存孙毅劈斩的时候不一样。那次是一条线，从肩到手，干净利落。这次是整个身体都在走，从脚趾到头顶，气机的路线比那一招深得多、密得多。

不舒服。但满了。

不是人的招。是怪物的。

他抬头看向一百多米外那个站着的轮廓。

有个问题。一个他来不及想的问题。它冲的是姜北，为什么他录上了？来不及想。存上了就够了。

下面的事是用它。

【第二十七章回顾】
梗概：新一波猛攻到达。三类数量激增至八只以上，夹杂大量二类。姜北正面连续扛多只三类，体力和气机消耗加剧——呼吸从鼻息变成嘴巴出气。马大壮备用棍裂纹扩大，每一棍都手抖。沈擎膝盖被二类撞倒磕伤，跑会疼。窄街方向放了一只三类。变异三类再次蓄力冲撞姜北——沈擎站在五六米外完整看清了四秒蓄力全过程：脚趾→踝→膝→大腿→脊椎弓→骨板前倾→肩胛挤压→四肢弹射，一条完整的力的路线。定桩极限加剧：整片路面炸开，膝盖弯了一个明显角度，上半身也晃了。马大壮趁变异三类被弹回的间隙全力一棍砸在骨板上——只留白印，二阶的力穿不透。沈擎心里算账——好碟，但它冲的是姜北，按已知规则录不上。变异三类第二次蓄力冲击时回响意外激活。气机路线比孙毅劈斩深得多密得多，从脚趾到头顶全身在走。沈擎察觉到矛盾——它冲的是姜北，为什么录上了？来不及想。手里有东西了。
伏笔：①回响激活矛盾——关键埋设。变异三类冲击目标是姜北，沈擎在侧面五六米，按已知规则（攻击意图指向自己）不该录上。沈擎已察觉但来不及深想。留给第39章唐遇质疑、第二卷展开。②回响录入非人类攻击——首次。气机路线更深更密更不舒服，整个身体都在走，与复制人类招式（孙毅劈斩只走肩到手一条线）形成对比。③姜北定桩极限升级——膝盖弯了明显角度，上半身也晃了，地面碎到“兜不住了”。脚下凹陷比路面低八九公分。再接一次可能扛不住。④变异三类攻击模式完整记录——蓄力四秒，力的路线从脚趾到弹射，本能不变。为第28章沈擎使用回响复制冲撞提供基础。⑤重锤无效——马大壮全力一棍砸变异三类骨板只留白印，建立“二阶穿不透”的认知。为第28章裂纹扩大后重锤打穿做铺垫。⑥“好碟”呼应——第11章“好碟放不进去”、第19章“好碟录不上”，本章“好碟”终于录上了，但录上的方式违反已知规则。
状态变化：回响从空仓→满仓（变异三类冲撞）。录入时全身气机路线走了一遍，体感不舒服但完整。姜北定桩严重消耗，脚下凹陷近九公分。马大壮备用棍裂纹继续扩展，且确认打不穿变异三类骨板。沈擎膝盖磕伤，补剂用了一支剩十支。变异三类站在一百多米外，已冲撞两次，骨板上两道浅痕但仍完好。
下章衔接：第28章“用它自己的招打它”——沈擎出手。复制变异三类冲撞，用人的身体和C级气机施展。效果不如原版但找到角度——撞在侧腹裂纹处，裂纹扩大。马大壮「重锤」完整展示（不可能性锚定）：力锁在接触面上不扩散，硬币大小的点承受全部力量。姜北正面踹进裂口。三人配合击杀变异三类。
