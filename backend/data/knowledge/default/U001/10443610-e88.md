# 第二十四章：城南防线第一夜

唐遇闭上了眼。

不是在听。沈擎站在他身后五米，看得很清楚：唐遇把耳机从左耳摘了下来，攥在手里，两只手手指微张，像在摸一面看不见的墙。

这时候战场上什么声音都有。南边省道上那群东西逼过来的脚爪刮柏油路面，嘶吼，很多只同时嘶吼，频率不一样，混在一起像金属片在搓。远处其他防区有爆裂声传过来，闷的，一声接一声。风把垃圾桶盖掀了，铁皮在地上滚。

吵。耳朵嗡嗡的。

唐遇开口了。

“正面方向，八百米。第一梯队二类，三十七只。前锋速度每秒四米，最快的那只偏右侧排水沟，五米二。后方六百米处三类两只，间距十二米，跟在二类群后面。再后面的信号太密，暂时分不开。”

三十七只。偏右侧排水沟那只最快，五米二。三类两只，间距十二米。

这不是在听。

临渊镇民兵队有个老侦察兵，耳朵好使，能听出异种在哪个方向、大概多远。但那个“大概”的精度是“在左边”“不太远”。

唐遇报的是方位、数量、速度、间距。

风声盖着嘶吼声盖着铁皮滚地声，整条街吵得人说话得扯嗓子。唐遇两只耳朵一只摘了耳机一只没戴，站在那里手指微张，三十七只，五米二，十二米。

跟耳朵没关系。

“五百米。”唐遇睁开眼，语速又快了一档，“二类群开始散开，前排十一只宽度覆盖整条路面，后排跟进。排水沟里六只，沿沟走，比路面的快。三类没变速，跟在后面。”

姜北站在丁字路口正中。

他没回头。两脚碾了一下地面，柏油路面发出细微的碎裂声，两只脚的前掌往下沉了一点，像树根往泥土里钻。

“三百米。”唐遇说。

沈擎看到了。

路的尽头。灰褐色的低矮身影从暮色里涌出来。贴着地面，四条腿，背上短刺，一只、三只、七只、十几只，是一片。路面上一层，排水沟里一层，两边废弃平房的院墙上还爬着几只。

但主要的，确实在路面上。确实在排水沟里。

水往低处流。

“一百五十米。”

两只脚的鞋底已经陷进柏油路面大概两公分。

握着短刀退到姜北右后方。机动位。视线在三个方向之间切：正面的二类群、西侧窄街入口、姜北的后背。

第一只二类冲到五十米的时候加速了。四条腿蹬地的频率突然变快，嘴部裂开，露出灰白色软体结构，嘶声尖利。

后面跟上来一片。

“接触。”唐遇说。

姜北没动。

第一只二类扑上来的时候姜北一拳打出去。干净，短，从胯部发力，拳面带着一层薄薄的气机光泽。二类被打飞出去四五米，在地上翻了两滚不动了。

第二只。第三只。第四只从侧面绕。姜北左肘横扫，第四只撞在肘上弹出去，嘴里喷了一片灰色液体。

不难。二类对三阶来说不难。

但量大。

五只、八只、十二只。正面涌过来的速度比姜北清理的速度快。有两只从姜北两侧溜过去了。

迎上去。

短刀正握，C级气机附在刀刃上，薄薄一层，不够硬，但二类的皮不厚，够了。第一只从左边窜过来，他侧身让过扑咬，刀从下往上划，划开腹部。第二只从右边来，踢了一脚把它踹偏方向，刀尖捅进背脊短刺之间的软肉。

两只。

第三只他没来得及处理。马大壮的棍子从侧面横扫过来，二类飞进旁边的废弃院子，撞在墙上，墙皮掉了一片。

马大壮站在西侧窄街口。棍子竖在身前，胶带缠的那截鼓出来的地方正好在手握的位置。

“窄街没东西。”马大壮说。声音大，频道和嘴巴同时在说，“都从正面来的！”

正面。姜北一个人在扛。

回头看了一眼。姜北站在路中间，面前已经倒了七八只二类。活的还在涌。有一只三类混在二类群后面，体型明显大一截，四足，肩高到姜北胸口，扁平头，甲壳在暮光里泛灰。它没冲，站在后面等着。

“唐遇，三类位置。”

“正面偏左，九十米。另一只一百二十米，偏右。都没加速。”

没加速。二类在前面耗体力，三类在后面等。

头一波是试探。日志上写的。

正面已经打了快两分钟了。二类的尸体在他脚前堆了小半圈。频道里传出来的底噪稳定，没有喘。三阶打二类，体力消耗不大，但气机在慢慢磨。

又有三只从姜北两侧漏过来。沈擎杀了一只，另两只往北边跑。

“北边两只过去了。”

“我扫到了。跑了，往收费站方向。二组会接。”

短刀上沾了灰色的液体，粘稠，有腥味。沈擎在裤腿上蹭了一下，回到机动位。

正面的二类群在变薄。那两只三类还是没动。

“二类前锋清得差不多了。”唐遇说，“后续补充速度在降。三类——”

话没说完。

左边那只三类动了。

抬头的时候它已经在冲了。四条腿蹬地，甲壳前倾，速度比二类快一倍不止。正面直冲姜北。

姜北的脚碾了一下地面。

三类撞上姜北的那一瞬间，沈擎看到了一件事。

它体重至少两百公斤，加上冲刺速度，这一下能把一面砖墙撞穿。扁平头正面撞在姜北双臂交叉格挡上。

姜北的上半身没动。

不是“扛住了”。人扛冲击，不管多强壮，身体一定会有一瞬间在吸收力——脊椎弯一下，肩膀缩一下，重心晃一下。

姜北没有。

沈擎就在侧面五米。他看得很清楚：姜北的小腿从膝盖往下绷得像两根钢柱，但膝盖以上，腰、背、肩膀，是松的。不是松懈的松，是力根本没有到达那里。

冲击力从三类的头部传到姜北的手臂，然后拐了个弯，沿双腿直接灌进了地面。

柏油路面炸开了。蛛网状裂纹从他两只脚往外扩了一米多，碎块弹起来打在三类甲壳上啪啪响。但姜北的上半身一动没动。右拳打出去的时候，躯干稳得像什么都没发生过。三类被弹回去三米远，四只脚在柏油路上刮出四道白印。

定桩。

不是“站得稳”。是力的方向被改写了，冲击力不走身体，直接导入地面。

这不是人能练出来的东西。

三类晃了一下，重新蓄力，第二次冲。姜北又接了。脚下路面碎了更大一片，三类被弹出去五米多，甲壳上多了一道浅痕。

右边那只三类也动了。

“右侧三类起步，六十米。”唐遇说。

“大壮，正面来了。”沈擎在频道里喊。

马大壮从窄街口冲出来。棍子扛在肩上，跑起来地面震。

右侧三类冲过来了，比左边那只快，甲壳前端低压，冲撞姿态。它冲的是姜北的侧面。

“我来。”马大壮。

棍子从肩上抡下来，对着三类的前腿。冲刺时前腿伸出那一瞬间膝关节外露，时机卡得准。一棍。三类前腿被砸偏了角度，冲刺姿态散了，身体往右歪，甲壳擦着姜北肩膀滑过去，撞断了路边一棵水泥电线杆。

马大壮追上去补了一棍，砸在甲壳上。白印。甲壳太硬，二阶的力穿不透。

但它被打乱了。姜北转身一脚踢在侧腹甲缝。三类嘶了一声，蜷缩，转身想跑。

沈擎从侧面绕过来堵退路。没硬冲。三类转身时后肢蹬地有顿挫，转身慢，这一两秒它顾不上侧面。

不需要杀它。挡住就行。

三类犹豫了不到一秒——侧面一个人挡路，正面姜北逼近，后面马大壮举着棍子。三面堵死。

姜北的拳头落在甲缝上。

三类倒了。

左边那只也差不多了。姜北打了四拳，最后一拳在扁平头侧面，歪倒在地上，腿越蹬越慢。

沈擎站在原地喘了两口气。短刀尖朝下，手臂酸。

频道里马大壮的声音传过来，大，喘，亮堂：

“沈擎你他妈还真说准了！一只都没从侧面来！”

没接话。他在补位回姜北右后方的路上，脚底踩到一只二类的尸体，滑了一下。

“安静。”姜北说。

频道安静了。

姜北站在路中间。脚下的柏油路面碎了一圈，碎块和灰色血液混在一起。呼吸比刚才重了一些，但只是一些。旧款防护服的袖口沾了灰色液体，灰黄的面料上看不太出来。

走回机动位站定。风还是南风，渊味没散。

他低头看了一眼短刀。刀刃上有个小缺口。不影响用。

唐遇的声音在耳机里响起来。平，稳，快。但这次他顿了一下。

“第二波。正面方向，一公里外。”

顿了一下。

“比刚才多。”

【第二十四章回顾】
梗概：异潮第一波到达城南防线丁字路口。二类群三十余只打头，沿路面和排水沟涌来，三类两只混在后方。唐遇闭眼展开「声呐」，摘掉耳机，在噪音覆盖的战场上报出精确到方位、数量、速度、间距的数据——感知的不是声音，是生命体的气机波动。姜北「定桩」正面接住三类全力冲撞——冲击力没有经过他的身体，沿双腿直接导入地面，柏油路面炸裂但上半身纹丝不动。沈擎在侧面五米亲眼看清了：这不是“站得稳”，是力的传导方向被改写了，不是人能练出来的。三组配合击杀两只三类和大量二类。丁字路口卡位完全生效——正面单一来路，西侧窄街无异种进入，北侧撤退通道保持畅通。马大壮频道确认“你他妈还真说准了”。第一波结束后唐遇报告：第二波正面一公里外，比刚才多。
伏笔：①唐遇声呐——不可能性锚定完成。战场噪音中摘掉耳机依然精确报数，感知的是气机波动不是声音。②姜北定桩——不可能性锚定完成。力的传导方向被改写，冲击力不经过身体直接导入地面。沈擎从“以为是站得稳”到“理解力没有经过他”，认知跃迁。③丁字路口卡位——兑现。第22章战术讨论的全部判断得到实战验证（单一来路、窄街卡三类、撤退通道畅通）。④“水往低处流”——再次兑现。异种确实沿路面和排水沟涌来。⑤回响空仓——保温。整个第一波沈擎用短刀和C级气机补位，没有回响可用。⑥姜北穿旧款防护服——视觉保温。“灰黄的面料上看不太出来”。
状态变化：第一波清理完毕。丁字路口卡位验证成功。唐遇声呐和姜北定桩的“不可能性”被沈擎亲眼确认。短刀刀刃有小缺口。回响仍空仓。第二波即将到达，规模更大。
下章衔接：第25章“打到第几波我已经记不清了”——持久战。沈擎发现变异三类的指挥行为，提出战术判断。唐遇预震①（“都往这边来了”）。
