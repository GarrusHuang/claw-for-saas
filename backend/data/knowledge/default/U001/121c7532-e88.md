# 第十三章：我抄你的作业你介意吗

哨响。

孙毅没动。

他站在对面十二米开外，两手垂在身侧。一组其余五人散在侧翼和后方，但没人有上前的意思，对面就四个人，而且对位抽签的规矩摆着，先打对位。

训练场二楼的模拟对抗场不大，两边各有一条白线当防线，中间是十五米左右的空地。软垫铺了一圈，顶上的日光灯管嗡嗡响，有一根闪。

沈擎站在三组防线前面。姜北在他身后三步远的中路，马大壮左翼，唐遇最后面。

四对六。防守方。三分钟。

孙毅活动了一下脖子，咔吧一声脆响。

“我轻点。”他说。

不大声，但场子不大，两边观战的人都听见了。二组几个人对视了一眼，没接话。赵姐在场边低头写字，笔没停。

沈擎没回嘴。

他在看孙毅的脚。

左脚比右脚靠前半步，重心偏后。上次在新华路十字路口，他在外围隔了四十多米看的时候就注意到这个习惯，孙毅出招之前，左脚会先外撇。

不是每个人都有这种起手信号，但孙毅有。

三秒。五秒。

孙毅动了。

他走过来的。步子不快，甚至有点散漫。走到中间线的时候停了一下，量了一下距离。

然后他提速了。

左脚外撇。

沈擎全身的注意力收到一个点上。

气机劈斩。

他见过这一招。昨天在新华路，隔着四十米，看得清清楚楚。右臂发力，肩带动肘带动腕，气机从后背走肩走到指尖，外延半米，凝成一道窄锋利的弧线。

但隔着四十米看和正面挨是两回事。

孙毅右臂抬起的瞬间，沈擎的身体比脑子快——侧身。卸。临渊镇民兵对付冲过来的东西有一条土经验：正面扛不住就别正面扛，把身子转四十五度，让力从身侧滑过去。

气机劈斩擦着他左肩过去。

擦。

不是完全擦过去。

左肩上的训练服裂了一道口子，从肩头到锁骨下面，布料翻开。里面的皮肉翻起一层，不深，但见红了。血珠子沿着锁骨往下淌，滴在地面软垫上。

疼。

烧着疼，刀片烧红了划上来的那种。

沈擎踉跄退了两步。左脚踩在白线边缘，差点出界。

训练场里有人吸了口气。

孙毅收招站定，目光扫过沈擎。那种从上往下的眼神，带着确认的意思。确认对手没什么花样了。

“还站着？”他问。

语气倒像夸他。

马大壮在左翼攥紧了拳头。姜北没动。唐遇在最后面，眼睛闭着，「声呐」开着，但这会儿不需要报点，他在听。

场边的人在嘀咕。赵姐笔停了一下，又继续写。

沈擎低头瞟了一眼左肩。血已经渗透了训练服的布料，颜色发暗。肩膀上的伤口不碍事，皮肉伤，没伤到骨头。但疼感顺着锁骨蔓延到了整个左半边身子。

他抬头。

有东西了。不是肩膀上的伤。

孙毅那一刀是冲他来的。不是演戏，不是过招的意思。那道气机劈斩划到他身上的时候——角度、力度、速度——都是冲着他来的。

演练归演练，那一下是实打实的。

脑子里原本是空的，空了一个月。自从打完那个二类之后就一直空着，像个拔了碟片的老式DVD机。

现在有东西转进去了。

不是模模糊糊的轮廓。是完整的——右臂发力轨迹，肩到肘到腕到指尖，气机凝聚的形状，左脚外撇角度，甚至劈出去那一瞬间空气被撕裂的声音。

全的。

上次在新华路看了个全场，什么都记住了，但回响里什么都没有。

因为不是冲他来的。

今天是。

沈擎站直了。

左肩的血还在淌，他没去擦。

他换了个姿势。

这个动作很小，场上大部分人没注意到：他的左脚往外撇了一点。

孙毅注意到了。

但来不及了。

沈擎的右臂抬起来。肩带动肘，肘带动腕。气机——他只有C级的气机——从后背涌上肩头，顺着手臂往下走，到指尖的时候只凝出了薄薄一层。

不够。

跟孙毅那刀比起来，差了不止一个等级。C级的气机量撑不起三阶的招式。

但够了。

气机劈斩。

从沈擎手里打出去的时候，弧线只有孙毅那道的一半宽、一半长。但轨迹一模一样——发力点、传导路径、劈出的角度。

甚至左脚外撇的那个微小习惯。

一模一样。

弧线没往孙毅身上招呼。沈擎收了一下，落点偏低，砸在孙毅面前两步远的地面上。

软垫炸开了。

不是切开的，是砸开的。C级气机不够利，但动能在，裹着气机的弧线拍在地面上，软垫底下的水泥地面崩出一个浅坑。碎石子和软垫碎片崩出来，溅了孙毅一裤腿一鞋面。

孙毅退了一步。

不是被打退的。是他自己退的。本能的、身体先于脑子的那种后退。

然后他站住了。

训练场安静了。

彻底安静。连那根闪的灯管都像挑了这个时候消停了。

三十来号人，没一个出声的。

二组组长张着嘴。后勤老周的文件夹掉地上了，他弯腰去捡的动作停在半路。赵姐的笔尖戳在本子上，墨水洇出一个圆点。

马大壮在左翼。他的拳头松开了。傻了。

唐遇睁开了眼。

姜北站在原地，一动没动。他的表情没变化，或者说，看不出变化。但他的手指在裤缝上动了一下。

沈擎甩了甩右手。

手腕有点酸。C级身体硬扛三阶招式走了一遍，不好受。手臂肌肉在跳，从肩膀到手指尖都在发颤。但这是事后反应，出招的时候没感觉。

他看了孙毅一眼。

孙毅表情很好看。不是好看的那种好看，是值得记一笔的那种。嘴角弧度还在，但下面绷着的东西变了。原来是轻的，现在不轻了。

他退了一步。刚才那一步。

演练场上退一步不算什么。但孙毅退了一步，在所有人面前。一组几个人站在他身后，表情各异，有懵的，有皱眉的，有看沈擎的，有看孙毅的。

沈擎的左肩还在流血。训练服从肩头裂到锁骨，暗红的血把左半边衣服洇湿了一片。

他拽了一下裂开的布料，没拽动。算了。

场边有人开始窃窃私私。声音不大，一小团一小团的，在角落里。

赵姐低头看了看本子上那个墨点，翻了一页，开始写。

老周终于把文件夹捡起来了。他看了看两边，清了清嗓子，但喊出来的声音有点劈：

“对位结束。其余人员进入——呃——进入对抗。”

一组和三组剩下的人往场上走。但所有人的眼神都从沈擎和孙毅身上绕了一圈才收回去。

后面的对抗正常打了。一组人多，攻势猛，三分钟到的时候三组防线没破但也摇摇欲坠。姜北扛了主力进攻，马大壮左翼拼了一轮，唐遇的报点精准但也只能精准。

但已经没人在意结果了。

散场的时候人群三三两两往外走。沈擎经过二组那片的时候，有个人，他不认识，应该是二组的外勤。那人朝他点了下头，嘴动了一下，没说话。

马大壮追上来，一巴掌拍他后背。

“操！”

就一个字。但拍得很响。

沈擎被拍得往前一趔趄，左肩的伤扯了一下，嘶了一声。

“轻点。”

“对对对，忘了忘了——”马大壮搓了搓手，“你没事吧？肩膀那个——”

“皮肉伤。”

唐遇走过来。他什么都没说，从口袋里掏出一包纸巾递过来。

沈擎接了，撕开一张，摁在肩膀上。纸巾沁红了一块。

姜北走在最前面，没回头。

出训练场的时候，沈擎余光扫到人群后面有个人掏出手机，像是在看时间，又像不是。

那人穿着清源分局的训练服，但沈擎不记得见过他。

---

回值班室的路上，马大壮的嘴就没合上过。

“他退了！你看见没有？他退了一步！”

“全场都看见了。”唐遇说。

“那一招是不是跟他那个——一模一样？我站在侧面看的，出手那个角度——”

“差不多。”唐遇说，“力道差很多。”

“力道不是重点！重点是一模一样！他三阶的看家本事，你一个C级——”马大壮搓着手，“操，怎么做到的？”

他看沈擎。

沈擎拿纸巾摁着肩膀，没理他。

“你别问。”姜北的声音从前面传来。他还是没回头，“回去处理伤口。”

马大壮嘴巴张了张，看了看姜北的背影，又看了看沈擎。

咽回去了。

值班室里，沈擎坐在椅子上，唐遇帮他把训练服左边剪开。伤口不深，但长，从肩头一直划到锁骨下，一条红色的线。

唐遇拿酒精棉片擦的时候，沈擎吸了口气。

“你说你轻点。”

“我没使劲。”唐遇面无表情，“酒精本身就疼。”

马大壮蹲在旁边递纱布。“你说孙毅现在什么心情？”

没人接话。

马大壮自己答了：“憋屈。肯定憋屈。”

“你闭嘴。”姜北从门口说。他刚出去接了个电话回来，脸上看不出什么，“别在外面瞎说。”

“我在值班室说的——”

“值班室也别说。”

马大壮终于闭嘴了。

姜北看了沈擎一眼。目光在他肩上的伤口停了一下，移开了。

“明天照常出勤。”他说，“演练的事，谁问都别多讲。”

说完进了里屋。

门关上了。

马大壮用口型对沈擎说了三个字：“牛逼啊。”

沈擎低头看了看自己的右手。

手还在抖。细微的、肌肉不受控的那种抖。从手指尖到手腕，一阵一阵的。

用出去了。

脑子里那张碟，放了一次，空了。

又是空的。
