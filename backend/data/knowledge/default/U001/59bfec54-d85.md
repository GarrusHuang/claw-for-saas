# 第十八章：孙毅请客有什么目的

孙毅请客。

消息是马大壮带回来的。下午从城北管网出完勤回分局，他在走廊碰见一组的小赵，小赵说孙哥让他传话，今晚请三组吃饭，就在分局对面那家湘菜馆，六点半。

马大壮进值班室的时候脸上的表情很复杂。像捡到钱包但里面可能有炸弹。

“孙毅请咱吃饭。”

唐遇的手指停在终端屏幕上。

沈擎从行军床上抬起头。“你再说一遍。”

“孙毅请三组吃饭。”马大壮把这句话嚼了一遍，还是觉得不对味，“湘菜馆，六点半。”

值班室安静了几秒。

唐遇继续翻他的数据。沈擎重新躺下去。马大壮站在门口，等了一会儿没等到反应，急了：“你们不觉得奇怪吗？”

“奇怪。”唐遇说。

“那去不去？”

沈擎盯着天花板。天花板上那块蛤蟆形状的水渍好像又大了一圈。

“去。”他说，“白吃的饭不吃，亏。”

姜北五点四十回来的，从钱旺办公室出来。脸色跟进去的时候一样，看不出聊了什么。马大壮把请客的事说了，姜北想了两秒，点了下头。

“去。”

六点二十，四个人穿过马路，进了湘菜馆。

小馆子，两层楼，一楼大厅摆了十来张桌子，油烟味混着辣椒味糊在脸上。孙毅订的包间在二楼，推门进去的时候他已经坐着了——靠窗位置，面前摆了一壶茶，自己倒了一杯在喝。

看见三组进来，孙毅站起来了。

“来了来了，坐。”

沈擎打量了一下。孙毅穿的便装，深色外套，不是平时在分局那副样子。脸上的表情也不一样——不是走廊上碰到时那种嘴角往下拉的劲儿，带着笑，不过笑得有点生硬，像是对着镜子练过。

马大壮坐下来拿起菜单。姜北坐在孙毅对面，沈擎挨着姜北，最靠门口的位置。

“随便点。”孙毅把菜单推过来，“这家剁椒鱼头不错，上次一组来过。”

点菜的时候没人客气。剁椒鱼头、农家小炒肉、干锅花菜、一份酸辣土豆丝、一份手撕包菜。孙毅又加了个辣子鸡和一盘毛豆。

“喝什么？啤的白的？”

“啤的。”姜北说。

孙毅叫了一箱雪花。

菜还没上，孙毅先给每人倒了杯茶。壶里的茶已经泡得发黑了。

“最近忙坏了吧？”孙毅先开口，“我看三组的出勤表，十月排得比我们一组还满。”

“差不多。”姜北端起茶杯，没喝。

“我们北线也多了。二类比上个月翻了一番。前天晚上连出三趟，小赵跑到最后腿抽筋了。”孙毅笑了一下，“年轻人不行。”

马大壮接话：“我也抽筋了，膝盖那儿，蹲不下去。”

“你那是太沉了。”孙毅拍了拍自己的肚子，“减减肥。”

马大壮哼了一声。

啤酒开了，碰了个杯。孙毅喝了大半杯，擦了擦嘴。

“城南那边情况怎么样？我听说监测桩出了点问题。”

姜北放下杯子。“出了点状况，钱局在处理。”

“严重吗？”

“不清楚。技术口在查。”

孙毅“嗯”了一声，没再追。菜上了。剁椒鱼头端上来，辣椒的烟呛了马大壮一下。他先夹了鱼腮边的肉，又去搅鱼头里的软骨，嚼了嚼吐在骨碟里。唐遇看了一眼那坨东西，把自己的骨碟默默挪远了两公分。

沈擎夹了两筷子小炒肉，没说话。

孙毅吃了几口，又起了个话头。

“沈擎，你老家临渊镇的？”

沈擎嚼着肉。“嗯。”

“那地方离渊口多远？我记得不到三十公里。”

“二十七。”

“离得那么近，小时候能看到裂隙吗？”

“能。天黑的时候从镇上就能看见光。”

孙毅摇了摇头。“你们那儿的人胆子真大。换我早搬了。”

沈擎没接这话。他拿筷子戳了一块鱼头上的辣椒，把下面的肉挑出来。

孙毅喝了口酒，眼睛往桌面上落了一下。

“对了，之前省局来检查，那个周——”他打了个响指，做出想名字的样子，“周鹤？督查处的？”

“副处长。”姜北说。

“副处。”孙毅重复了一下，“带了几个人来？”

“大会上露面的两个。其他的不清楚。”

孙毅“哦”了一声，端起酒杯转了转。“他来查什么呢？例行检查还是——”

“省局的事。”姜北说，“你问钱局。”

孙毅笑了。“随便聊聊。这不大家都好奇嘛。省局的人来了之后没几天就开始忙了，有人说跟那次城中心的三类有关，有人说跟监测桩有关……”

他停了一下，看了一圈桌上的人。马大壮在埋头吃。唐遇在啃毛豆。姜北端着杯子没喝。

沈擎在夹鱼。

“也不知道查完没有。”孙毅又补了一句。

姜北没回答。

筷子夹着鱼肉的沈擎抬了一下头。

“孙哥。”他的语气很随意，顺嘴接了一句，“你们一组十月的处置数据交了没？”

孙毅看过来。

沈擎把鱼肉送进嘴里，嚼了两下。“我听赵姐说，省局下了通知，要交叉比对各组的出勤记录和处置报告。十月的先交。”

桌上安静了一拍。

马大壮还在吃，但筷子慢了一下。唐遇剥毛豆的手没停。姜北什么反应都没有，端着杯子喝了一口啤酒。

孙毅的筷子停了。

不到一秒。然后他夹了一块辣子鸡，笑着说：“交了交了，上个礼拜就交了。”

他把辣子鸡塞嘴里，嚼了两下，又说：“省局现在事多，表也多。我们那边小赵天天在填表，比出外勤还累。”

话题滑开了。

后面聊了些有的没的——补剂什么时候发、明年的编制名额、冬天的出勤补贴是不是还跟去年一样。孙毅聊得挺自然，没再往督查处和监测桩上绕。

七点四十，孙毅叫了单，四百三。马大壮喝了两瓶啤酒，脸有点红。唐遇全程就动了一杯。

分头走。三组过马路进了旁边的巷子。马大壮回头确认孙毅走远了。

“他到底想干什么？”

姜北的脚步没停。“他在试探。”

“试探什么？”

“我们知道多少。”

“那他知道什么？”

“他知道省局来了人。知道监测桩出了问题。知道城南最近不太平。”姜北说，“但他不知道钱局跟我们说了什么，不知道周鹤来查的是什么。他想知道。”

马大壮“嗐”了一声。“那他问也白问，咱也不知道。”

姜北没接这句。

走了几步，他回头看了沈擎一眼。

没说话。就是看了一眼。

沈擎把两只手揣进兜里。

夜风带着那股烧焦铁锈的味道，从南边过来。他吸了一口，鼻腔涩涩的。

他在算一笔账。

四百三。孙毅花了四百三，得到了什么——姜北的三句废话，马大壮的饭量展示，唐遇的半壶毛豆，以及沈擎那句“交叉比对”。

前三样不值钱。最后那句值多少？

孙毅在城中心联合处置的报告里把唐遇声呐定位的功劳写成一组自行完成。沈擎不确定省局是不是真要交叉比对——但孙毅不敢赌。调度台通讯时间戳上记着声呐接入时间，报告里没写，时间戳上有。

他今晚会翻那份报告。翻了改不了。但他会睡不好。

四百三，买了一个睡不好的觉。

亏的是孙毅。

沈擎松了松肩膀。左肩的伤好得差不多了，晨练的时候已经不怎么扯着疼了。

回到分局楼下，马大壮和唐遇先上去了。沈擎去车棚取充电宝。

路过姜北宿舍的时候，门没关严。走廊的灯照进去一条光缝。他没刻意看。但余光扫到了。

姜北坐在床边，手里拿着一张照片。

不大，老式冲洗的那种，四寸或者五寸。姜北拿着，角度看不清正面。但他坐姿不对——不是平时随时能弹起来的绷劲，整个人陷在床沿上，肩膀往前塌着。

像扛了一天的什么东西，终于在没人看见的时候放下来了。

沈擎没停步。脚步声没变，速度没变，走过去了。

他从车棚里拿了充电宝，原路回去。再经过那扇门的时候，门合上了。

上楼，进值班室。

马大壮靠在椅子上打嗝。唐遇在刷牙。

沈擎把充电宝插上，躺到行军床上。

他盯着天花板看了一会儿。那块蛤蟆水渍在灯光下没有白天明显。

姜北手里那张照片是谁？上次训练场被拒那天回来，他看到姜北摸过抽屉——大概就是那张。

他没想问。有些东西，问了人家得回答，回答了就得交代，交代了就变成重量。

值班室的灯关了。马大壮的呼噜在三分钟后准时响起来。

明天早上六点，天台。

【第十八章回顾】
梗概：孙毅反常请三组吃湘菜馆，席间旁敲侧击试探城南异种活动频率、临渊镇背景、省局督查处来访细节。关键时刻孙毅追问督查处查什么，沈擎插话——"省局要交叉比对各组出勤记录和处置报告"，将第12章孙毅报告抢功的事变成暗雷。孙毅筷子停了一下，话题滑开。散场后姜北判断"他在试探"，回头无声看了沈擎一眼（认可）。沈擎算账：四百三买了孙毅一个睡不好的觉。夜里路过姜北宿舍，看到他坐在床边拿着照片，整个人卸了防——沈擎没停步，没问。
伏笔：①孙毅暗线——推进。孙毅主动请客试探=他对周鹤检查和城南情况有焦虑。沈擎的"交叉比对"让他不安（呼应之前抢功的尾巴）。②姜北心结——推进。照片第二次正式出现（第8章抽屉→本章手持），沈擎看到姜北在无人时刻的疲倦和卸防。③编外价值——微推进。沈擎一句话解围+反将，不靠拳头靠脑子。④临渊镇——保温。孙毅饭桌上提到渊口距离。
状态变化：回响仍空仓。孙毅从忌惮升级为不安（交叉比对的威胁）。姜北对沈擎的认可从"给实用信息"升级为"关键时刻接话解围"。沈擎主动选择不追问姜北的私事（关系边界）。
