# 第二十八章：用它自己的招打它

变异三类站在一百多米外。两道浅痕挂在骨板上，姜北两拳打的，跟指甲划痕差不多。

沈擎盯着它。

它也在看他们。竖瞳亮着，暗黄色的光在裂隙余照里一闪一闪。不急。它知道正面那个能接住冲撞的人在发抖，知道时间站在它那边。

但沈擎手里有东西了。

他在算。

C级气机。一百三十斤。骨头是人的骨头，肉是人的肉，肩膀撞上去跟三米多长、全身骨板覆盖的东西根本不是一个量级。回响的路线在体内铺着，热的，涨的，从脚趾到头顶。但路线是管道，气机是水。它的管道能灌一条河，他的C级灌进去连管壁都湿不满。

正面撞它？死。

不需要正面。

变异三类更大更重，转身比普通三类更慢。蓄力四秒冲刺三秒，冲完刹不住，转身至少两秒。

两秒。够了。

它冲的方向是正面。冲完之后被弹回来，四条腿在路面上刮着找抓力。那个窗口里侧面是空的。

侧腹。骨板缝隙。变异三类的缝挤得紧，刀捅不进去。但冲撞是面，不是点。C级气机灌进回响的路线，打出来可能只有原版三成的力。三成的力集中在肩膀撞上去的那块面积上，对准缝隙，够不够把缝挤开？

不知道。

但姜北再接一次可能扛不住了。他的双腿还在抖，从膝盖往下，细微的，不说别人看不出来。脚下那个坑已经深得快没过脚踝了。

没有第二个方案。

“姜哥。”沈擎在专频里开口。

姜北没回头。他在看正面。变异三类还没蓄力，站在远处，竖瞳盯着这边。

“它再冲一次。你接住。弹回来的时候别追。”

频道安静了一秒。

“你要干什么？”

“用它自己的招打它。”

又安静了一秒。姜北没问怎么打。没问录上了没有。没问C级气机够不够。

“大壮，听到了没有。”姜北说。

“听到了。”马大壮从窄街口那边应。

“弹回来之后等沈擎先上。他打完你跟。”

“跟什么？”

“你看到机会就知道了。”

马大壮没再问。棍子扛肩上，裂纹那条暗线在裂隙光里清清楚楚。

沈擎把短刀插回腰后刀鞘。这一下不用刀。刀是点，他需要面。

右手虎口破的那只，攥了攥拳，指节咔咔响。回响的路线在体内跟着动了，攥拳的时候气机从指尖往掌心收，热流顺着前臂往肘走。不是他在控制。路线自己在响应。

踩了别人踩过的脚印。脚印的深浅跟他的体重不匹配，但方向是对的。

从丁字路口往前走了十步。站在姜北右后方三米。

等它。

---

变异三类低下头了。

骨板缝隙张开。脊背前倾。脚趾抓地，四只脚的趾尖同时陷进柏油路面，隔着一百多米都听得到路面被碾碎的声音。

第三次蓄力。一模一样的起手。

踝锁死。膝压缩。大腿肌群在骨板下面鼓胀。脊椎从尾部弓起来，力的波沿脊柱往前传。肩胛骨之间缝隙张到最大。

弹射。

地在震。三米多长的身体贴着路面冲过来，骨板弧面在灰橙色光里拖出一条暗影，速度快到两侧的二类尸体被气浪掀翻了几具。

姜北蹬地。碾。两脚碾进坑底，坑底已经碎了好几层，碾不动了。小腿绷直，膝盖锁住。

撞上了。声音比前两次都大。整片碎裂区从姜北脚下往外扩，扩了快四米。碎块飞起来砸在路面上弹得到处都是。

姜北的膝盖弯了。

弯了一个很深的角度。大腿肌肉鼓起来，青筋从裤腿边缘冒出来。上半身往前倾了，第一次没稳住。

但他没倒。

牙咬着，一拳顶出去。

变异三类被弹回来。

这一次弹得没前两次远，姜北的力也不够了。四五米。四条腿在路面上刮，刮出又深又长的白印。还没站稳。侧面朝着沈擎的方向，整条侧腹暴露着。两秒。

沈擎的脚趾先动的。

不是他让它动的。是回响。路线在体内响了，第一个节点是脚趾，跟它一模一样。十个脚趾隔着鞋底抓地，力从趾尖往上走。踝关节锁。他感觉到踝骨被一股热流箍住了，骨缝里涨得发酸。

膝关节压缩。大腿肌群绷紧。C级气机沿着回响铺好的路线往里灌。管道是它的，水是他的。管道要一条河，他只有一桶水。水在管壁里走，填不满，到处是空的，但方向没偏。

脊椎弓起来。从尾椎开始，一节一节往上传。力的波沿脊柱跑的时候他听到自己脊背在响，咔咔的，骨节被力撑开又合上。疼。管道太大身体太小，骨头在被撑着。

肩胛骨之间的缝隙打开。胸腔里的气被力的波挤出去了一半，呼了一口短促的白气。

弹射。

两条腿蹬地的瞬间路面裂了一小片。比它蹬出来的小得多，但方向对，力的释放顺序对。脚趾→踝→膝→大腿→脊椎→肩胛→全部灌进冲刺方向。

他冲出去的速度比他跑过的任何一次都快。

风从两侧割过来。膝盖疼了一下，磕伤的那条腿蹬地时牵了一下。来不及管。

变异三类还在减速。四条腿刮着路面，侧面朝着他。竖瞳转过来了，它看到他了。后肢开始蹬地，要转身。

来不及。

十五米。十米。五米。他看到了侧腹骨板缝隙，两块骨板之间挤着的那条线，比普通三类的窄得多，但它在。

左肩前顶。

撞上去了。

肩膀砸在骨板缝隙上的瞬间，整条左臂都麻了。C级气机不够厚，缓冲层太薄，骨板的硬度直接传到肩关节上，拿肩膀去撞一面石墙的感觉。左边锁骨嘎了一声，不知道裂没裂。

但力的路线把所有气机集中在了接触面上。回响复制的不只是冲刺的力，是力在接触瞬间的运用方式，集中，不扩散。

骨板缝隙被撑开了。

不大。几公分。但沈擎感觉到了，撞上去的那一瞬肩膀往里陷了一点，触感从石墙变成了有弹性的东西。缝隙里灰色的软组织被挤出来一缕，热的，蹭在他防护服肩头上。

变异三类嘶了一声。不是之前的低频闷吼。是疼的。

它扭身。沈擎被侧面甩出去了。翻了一圈摔在路面上，左肩撞地，火辣辣的疼从肩膀窜到脖子。在碎石和灰色液体上滑了两米才停。

抬头看。

缝隙裂开了。侧腹两块骨板之间，原来挤得死紧的那条缝，现在开了一道口子，一拳宽。灰色液体从里面淌出来，骨板边缘翘着，露出底下深灰色的软组织。

“大壮。”趴在地上喊了一嗓子。

马大壮已经在跑了。

他等的就是这个。

那东西在转身。四条腿蹬地转向沈擎的方向，刚才那一下疼了，它要找让它疼的东西。

但马大壮比它快。

从侧面冲上来。备用棍举过头顶。两手握棍，右手在前左手在后，整个上半身的力量压在棍头上。

一棍砸在裂口上。

沈擎趴在五米外看到了。

棍头砸在裂口边缘骨板上的一瞬间，裂纹从接触点开始跑。

一条主裂纹，笔直的，从棍头落点往下，穿过整块骨板。分支从主裂纹两侧斜出去，一条、两条、三条，像树根一样往外扩。裂纹跑了半秒，整块侧腹骨板上布满了纹路。

但棍头周围一圈，没有。

完好的。棍头落点往外三四公分，骨板表面光滑如初，一道纹都没有。裂纹全从那一个点往远处跑。力没有往四周扩散，全部从一个接触点灌进去，沿着骨板内部纹理往最远处跑。

像敲碎了一个煮鸡蛋。蛋壳碎了一整面，但敲的那个点反而完好。

二阶的力气，全力一棍。上一次砸在完好骨板上只留白印。这一次同样大的力，锁在一个硬币大小的点上，全部灌进去了。

重锤。

它嘶吼了一声。侧腹整片骨板在裂纹间松动，几块碎片脱落下来砸在路面上。底下的软组织暴露出来一大片，深灰色的，在裂隙光里泛着湿润的光。

马大壮退了两步。手在抖，棍子还攥着但手指发白，指节一个一个突出来。二阶气机锁那么大一块面积，他把明天的力气借了一截。

备用棍从棍身中间断了。裂纹早就贯穿了，最后这一棍把它震成了两截。上半截飞出去砸在路边，下半截还攥在手里。

马大壮看了一眼手里的半截棍。没扔。

它在挣扎。侧腹大面积暴露，灰色液体从碎裂的骨板缝里涌出来。四条腿还在蹬，要跑。这次是真的要跑。竖瞳里那种从容没了，瞳孔在缩。

姜北到了。

从正面。三步。每一步踩在碎裂区上都带起碎石。他的腿还在抖，但三步跑完的时候抖停了。

右脚抬起来。

踹进裂口。

脚跟没入软组织的时候变异三类整个身体痉挛了一下。定桩。姜北脚底的力，从脚底往下灌的那个方向，改了。不往地面。往变异三类体内。

气机从脚底灌进去。

沈擎在五米外看到了结果。变异三类的侧腹从裂口处鼓了一下，从里面往外鼓。体内的什么东西被定桩的力冲碎了。

四条腿同时软了。

姜北单脚站着。右脚还踩在它体内。左腿独自撑着全身重量，膝盖在抖。变异三类侧翻下去的时候重量带着他的右脚往下拽，他踉跄了一步，脚从裂口里拔出来。鞋面上全是灰色液体，热气从液体表面蒸出来。

定桩不是用来站的。

变异三类倒在路面上。四条腿还在微微抽搐，骨板碎片散了一地。竖瞳的光灭了，暗黄变成灰白。

三米多长的身体横在路中间。灰色液体从侧腹裂口往外淌，在碎裂的路面上汇成一小滩。

从地上爬起来。左肩动一下疼一下，不知道是脱臼了还是骨裂了。膝盖也疼。整个人在抖，C级气机跑了一遍怪物力路线之后的透支反应。气机还有，但身体不行了，肌肉和骨头被管道撑过一遍，酸软得跟从水里捞出来一样。

走到变异三类旁边看了一眼。

三人打的。他撞开裂口，马大壮砸碎骨板，姜北从裂口灌进去。一个人干不了。

摸了一下左肩。锁骨那里鼓了一块，按下去刺痛。大概率没断，但肯定伤了。

亏不亏？

回响用掉了。手里又是空的。左肩伤了。膝盖磕的还没好。补剂剩……捏了捏腰包，十支。

换了一只变异三类。

盯着地上那具尸体看了两秒。灰色液体还在淌，热气还在冒。三米多长，骨板覆盖，竖瞳都灭了还能让人不想靠近。

不亏。

耳机里传来唐遇的声音。平，快，鼻音很重。

“它倒了之后二类在散。三类也在退。最近的异种信号……四百米以外。正面方向暂时没有在逼近的。”

指挥官倒了，底下的乱了。他那半对半错的判断里，另一半也验证了。

短暂的安静。省道上的风从南边吹过来，带着渊味，呛。远处裂隙的灰橙色光照着空荡荡的路面和两侧废弃平房。

马大壮走过来。手里攥着半截棍，另外半截在五米外。他看了看地上的变异三类，又看了看沈擎。

“你他妈刚才是拿肩膀撞的？”

“嗯。”

“你一百三十斤撞它三米多长？”

“对。”

马大壮看了他三秒。

“牛逼。”

“别夸了。”沈擎蹲下来，左肩动了一下，吸了口气。“算一下。变异三类没有单独补贴标准，只能按普通三类报。五百块。三个人打的，分不了。就算全算我头上也才五百。我这肩膀去医院拍个片子挂个号加上来回打车——”

“你闭嘴吧。”马大壮说。

姜北走到路边坐下了。

从开打到现在，不知道几个小时了，沈擎第一次看到姜北坐下来。之前所有间隙他都站着，脚碾在地里，随时准备接下一波。现在背靠着一辆废弃面包车的轮胎，两条腿伸直，膝盖不抖了，但大腿肌肉在跳，一下一下的，不受控制地跳。

唐遇靠在残存的电线杆底座上。鼻血干了一层又湿了一层，嘴角那道血痕颜色最深。手提终端搁在腿上，闭着眼，手指微张，声呐还没关。

“其他方向怎么样？”沈擎问。

唐遇没睁眼。“北线在撑。东线被突破了一个口子。二组那边有群众没撤完。”

东线。

耳机里调度频道传来钱旺的声音。

“三组。转向东线支援。”

沈擎闭了一下眼。

站起来了。

【第二十八章回顾】
梗概：沈擎使用回响复制变异三类冲撞。C级气机灌进怪物的力路线——管道太大水太少，力不够但路线对。他不跟原版比力量，找角度：等变异三类冲姜北被弹回的减速窗口，从转身盲区冲进去，左肩撞在侧腹骨板缝隙上。缝隙被撑开几公分。马大壮跟上——重锤首次完整展示（不可能性锚定）：一棍砸在裂口边缘，裂纹从接触点往外跑，布满整块骨板，但棍头落点周围一圈完好——力全部从一个点灌进去，没有扩散。像敲碎煮鸡蛋。整片侧腹骨板碎裂脱落。代价：手指发白，二阶气机透支，备用棍彻底断成两截。姜北最后跟上，右脚踹进裂口，定桩的力从脚底灌入变异三类体内——不是往地面，是往它体内。内部结构碎了。变异三类倒了。指挥官倒后二类散、三类退，正面暂时清空。三人在路边短暂休整。沈擎算账：回响用完又空仓，左肩伤了（锁骨疑似骨裂），补剂剩十支。变异三类按普通三类报只值五百块——“不亏”。姜北第一次坐下来了。调度命令转向东线支援。
伏笔：①回响复制非人类攻击的体感——C级气机跑怪物管道的具体差距感确立。路线对但力不够，“小号水泵打大号管道”。用完后身体发软——透支反应。②重锤不可能性锚定——完成。力锁在接触面不扩散。裂纹只从接触点跑，周围完好。“像敲碎煮鸡蛋”。二阶代价：透支，“借了明天的力气”。备用棍彻底报废。③姜北定桩攻击用途——“定桩不是用来站的”。力的方向改写不只能导入地面，能导入目标体内。代价极大（单脚支撑+要直接接触内部）。④三人配合击杀模式确立——沈擎撞开口子→马大壮砸碎外壳→姜北灌入终结。各自能力的分工定位。⑤回响空仓——再次空仓。⑥姜北坐下了——从开打到现在第一次坐下来。⑦东线被突破——转场钩子，接第29章。⑧马大壮备用棍报废——武器彻底没了，只剩半截。
状态变化：变异三类击杀。回响从满仓→空仓。沈擎左肩伤（锁骨疑似骨裂），膝盖仍磕伤，补剂剩十支。马大壮备用棍断成两截，手指透支发白。姜北定桩严重透支，第一次坐下。唐遇持续鼻血，声呐未关。城南防线正面暂时清空（指挥官死后异种散乱）。东线被突破，有群众未撤离。调度命令三组转向东线。
下章衔接：第29章“这个频道没人回话”——全局态势恶化。三组接令转向东线。钱旺拍桌亲自上前线。通讯闪断。唐遇预震②。
